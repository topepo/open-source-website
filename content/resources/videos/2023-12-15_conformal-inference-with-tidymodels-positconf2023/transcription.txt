<h3>What is a prediction interval?</h3>

<p><a data-t="50">All right, so maybe I should start first by saying, well, what's a prediction interval if you've never heard of that?</a> <a data-t="52">And so a prediction interval, it's like a confidence interval, but it's an interval on a separate type of object.</a> <a data-t="58">If you had like a 95% prediction interval, that means that you have a bounds where 95% of the time a new observation that you would acquire later will fall into that interval.</a> <a data-t="68">So whereas confidence intervals are on like the mean prediction, this is about new observations.</a></p>

<p><a data-t="72">And the little diagram here shows you both confidence interval, which is kind of very narrow for this data set, and a wider prediction interval.</a> <a data-t="81">And you can see there's a couple of data points on the bottom and a few on the top that don't fall in.</a> <a data-t="85">So I think that's a 95% interval right there.</a></p>

<h3>The intuition behind conformal inference</h3>

<p><a data-t="112">All right, so let's start with just some data, right?</a> <a data-t="115">So this is just like a histogram on its side.</a> <a data-t="116">It's 500 data points.</a> <a data-t="117">It's centered around zero, maybe like plus or minus like, I don't know, 0.15.</a> <a data-t="123">And so you spend some time collecting this data, and then you get the 501st data point, and let's say it falls down here.</a> <a data-t="130">And it's like, well, it's not outside the range per se, but is that like a new data point from the same distribution, or has something changed, like the model drift question and things like that?</a></p>

<p><a data-t="140">And so what we can do is, if we were from a statistical standpoint, if we want to make some sort of judgment about whether this new data point is from our original distribution, one thing we could do is use good old-fashioned quantiles.</a> <a data-t="153">So if you wanted to do like, let's say you wanted a 9% interval, what you could do is get the 0.05 quantile on the lower end and the 95% quantile.</a> <a data-t="163">And if you were to make a completely distribution-free probabilistic statement about that, you could say that 90% of the time, when I get data from this distribution, they're going to fall in between that interval.</a></p>

<p><a data-t="176">And so this is where the conformal part comes in.</a> <a data-t="179">So you would say that that new data point, if it fell within that interval, it conforms to sort of this original reference distribution here.</a> <a data-t="186">So if you were to take this data set, compute those particular intervals, so there's a lower 95%, and this is like the 95% interval quantile, and you would think that any new data point that falls in here would be consistent with the original data.</a> <a data-t="200">And you can see, of course, there's false positives, 5% of the time here and 5% of the time up here.</a> <a data-t="204">In this particular data point, you would not really consider to be consistent with the original data.</a></p>
