<h3>Machine readable filenames</h3>

<p><a data-t="94">The most basic thing you might want to do is use globbing to isolate files that match</a> <a data-t="99">a simple pattern.</a> <a data-t="102">A more sophisticated operation that comes up is the need to extract data from a filename</a> <a data-t="108">using a regular expression.</a> <a data-t="111">I'm showing a bit of R code that turns a bunch of filenames into a data frame, and I assume</a> <a data-t="117">you can do this in your preferred language.</a></p>

<p><a data-t="120">I use the underscore to delimit fields and the hyphen to separate words within fields.</a> <a data-t="126">I think it has to be this way because a very common field is a date and a hyphen is how</a> <a data-t="132">we separate year from month from day.</a></p>

<p><a data-t="137">The overall guiding principle is to make things easy for future you.</a> <a data-t="145">So a machine readable filename is easy to target with glob patterns and regular expressions,</a> <a data-t="151">and it's going to have a very intentional use of delimiters.</a></p>
