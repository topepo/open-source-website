<h3>Build on existing resources</h3>

<p><a data-t="40">So one is to build on existing resources. I used hex stickers from </a><a href="/software/rstudio/">RStudio</a><a data-t="40"> mainly. These</a> <a data-t="47">are licensed with the CC0 license. So I was able to use them and I found hexes from a</a> <a data-t="52">few other sources. And here's the list. It was sort of arbitrarily chosen.</a> <a data-t="60">And then you</a> <a data-t="60">need to use your skills. So once I had the hex stickers, I needed to array them in such</a> <a data-t="65">a way that it made a pattern that I could have printed on fabric. And you may not know</a> <a data-t="69">this about me, but I went to design school for a year. So I know how to use Adobe Illustrator.</a> <a data-t="74">And so I spent lots of time painstakingly moving these images around so that the left</a> <a data-t="78">hand side lined up perfectly with the right hand side and the top with the bottom. And</a> <a data-t="82">then I was able to upload this image to the website Spoonflower and get some custom fabric</a> <a data-t="87">sewn.</a></p>

<p><a data-t="93">And then the other skill that I used is that I know how to sew. And if you don't</a> <a data-t="96">sew, you might not know that there's a difference between two kinds of fabric. There's woven</a> <a data-t="101">fabric and knit fabric. Woven fabric is what like a regular button down shirt would be</a> <a data-t="105">made out of. It's not stretchy. It's not as comfortable. And you can sew it on a regular</a> <a data-t="109">sewing machine. Knit fabric is what they make T-shirts out of. It's stretchy. It's</a> <a data-t="113">comfy. It feels like you're wearing pajamas. But you need some special tools or skills</a> <a data-t="117">to sew it. So if you're going to work with woven material, you probably want to use a</a> <a data-t="121">regular sewing machine, which has one thing of thread and a single needle. If you are</a> <a data-t="125">going to work with knit fabric, you probably need a serger or an overlock machine.</a></p>

<h3>Data wrangling and sewing</h3>

<p><a data-t="127">And then I found that this quote, which I can't find the source for, but I remember</a> <a href="/people/hadley-wickham/">Hadley Wickham</a><a data-t="132"> saying something like, once you have the data in the right format, the</a> <a data-t="135">visualization is trivial. I found that once I had the pieces of the dress cut out, sewing</a> <a data-t="140">the dress was trivial. So getting everything so that it was right side up and was going</a> <a data-t="144">to line up nicely, that was the hardest part. And then I was able to just put it together.</a></p>

<blockquote><a data-t="132">once you have the data in the right format, the</a> <a data-t="135">visualization is trivial. I found that once I had the pieces of the dress cut out, sewing</a> <a data-t="140">the dress was trivial.</a></blockquote>

<p><a data-t="148">If you're using a serger, you use that to make seams. And if you look at the inside</a> <a data-t="152">of a T-shirt, like the RStudio Conf T-shirt, it looks like a nice bound edge. And then</a> <a data-t="157">I've got a picture of the seam from my dress there as well. The reason why it looks so</a> <a data-t="161">nice is the serger has a knife as part of it. So as you run the material through, it</a> <a data-t="165">cuts off the excess and then it binds it all together. This works for the inside seams</a> <a data-t="169">and to put pieces together, but it does not work for hems. So if you look at a professionally</a> <a data-t="173">made knit garment, like the RStudio Conf T-shirt, it has been used, they've used a cover stitch</a> <a data-t="178">machine or a cover hem machine to put it together. I don't have access to a cover hem machine,</a> <a data-t="182">so I had to do the hems on my regular sewing machine with twin needles. It didn't turn</a> <a data-t="186">out as nice as I would have liked.</a></p>
