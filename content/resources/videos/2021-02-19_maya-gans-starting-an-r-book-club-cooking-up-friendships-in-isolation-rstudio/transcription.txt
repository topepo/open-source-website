<p><a data-t="0">Hi, my name is Maya Gans, and in this talk, I want to tell you how to start your own virtual</a> <a data-t="12">book club.</a> <a data-t="12">So this idea came to me in the beginning of lockdown, I found myself feeling</a> <a data-t="20">really alone.</a> <a data-t="20">And also scrolling through Twitter, I saw everyone taking on these new exciting</a> <a data-t="26">hobbies.</a> <a data-t="26">Like remember the time when there were so many pictures of sourdough.</a> <a data-t="31">So I was</a> <a data-t="31">looking at my bookshelf, and I saw Advanced R.</a> <a data-t="36">And I realized that's a book that I use</a> <a data-t="36">as a reference, but I never read cover to cover.</a> <a data-t="42">So I wanted to use the lockdown opportunity</a> <a data-t="42">to do that.</a> <a data-t="47">And I found myself asking a lot of questions that I didn't have the answers</a> <a data-t="47">to.</a> <a data-t="55">So I reached out to John Harmon, the moderator of the R for Data Science Slack channel.</a> <a data-t="55">And</a> <a data-t="55">he allowed me to start a book club channel within that Slack and to kind of just see</a> <a data-t="59">what happens.</a> <a data-t="66">And the results were magical.</a> <a data-t="66">12 of us ended up meeting weekly for the entirety</a> <a data-t="66">of the book.</a> <a data-t="70">So I think over the course of 25 weeks, and we went from essentially a group</a> <a data-t="70">of strangers to now a group of really good friends.</a></p>

<h3>Why the book club worked</h3>

<p><a data-t="77">So first, I want to talk about the</a> <a data-t="77">cornerstones of why I think that this worked.</a> <a data-t="85">And the reasons are twofold.</a> <a data-t="85">We created a</a> <a data-t="85">microcosm within an already existing community.</a> <a data-t="91">So you might say there's already R ladies</a> <a data-t="91">or other chapters and groups out there.</a> <a data-t="98">And that's true.</a> <a data-t="98">But that's thousands of faceless</a> <a data-t="98">group members who, albeit, are extremely helpful and inviting.</a> <a data-t="104">Whereas this group was 12 of</a> <a data-t="104">us meeting on Zoom with the camera on and really humanizing this learning experience</a> <a data-t="111">with each other.</a> <a data-t="116">The other component was that we made this as asynchronous compatible as</a> <a data-t="116">possible.</a> <a data-t="122">So each meeting was recorded and put on YouTube.</a> <a data-t="122">So if you couldn't attend,</a> <a data-t="122">you can watch that.</a> <a data-t="127">But we also had this Slack channel for people to ask questions and even</a> <a data-t="127">answer them whenever they get to them.</a> <a data-t="131">So you weren't really limited in lockdown to</a> <a data-t="132">this single hour.</a></p>

<p><a data-t="141">Now, the meetings themselves kind of changed over time what we did with them.</a> <a data-t="148">In the beginning, we took the first 30 or so minutes to review what we found in the chapter,</a> <a data-t="154">and then use the rest of the time to answer questions.</a> <a data-t="154">But after a couple weeks, maybe</a> <a data-t="154">around week eight or so, I was getting kind of bored with this model.</a> <a data-t="161">And I wanted to make an</a> <a data-t="161">application.</a> <a data-t="167">I made a </a><a href="/software/shiny-r/">Shiny</a><a data-t="167"> app that uses the concepts within a chapter.</a> <a data-t="167">And granted,</a> <a data-t="167">this Shiny app was super buggy, didn't even work.</a> <a data-t="175">But within the hour, we all debugged it together</a> <a data-t="175">and got it up and running.</a> <a data-t="180">And I think that was kind of a paradigm shift where people started,</a> <a data-t="180">rather than just relaying information, they showed us their packages or other creations.</a> <a data-t="187">And it was incredibly inspiring and eye-opening to see how people are using these fundamentals</a> <a data-t="196">in these chapters.</a></p>

<h3>The secret sauce</h3>

<p><a data-t="205">So now let's get to the secret sauce of why I think this model worked, right?</a> <a data-t="211">We're not the first people to make a book club.</a> <a data-t="211">And I think it's because of two things.</a> <a data-t="211">One is</a> <a data-t="211">that we have a facilitator for each of our books.</a> <a data-t="218">And in this case, it was me, who, granted, I'm</a> <a data-t="218">pretty extroverted, and I was totally fine pinging people on Slack and being like, hey, it's Thursday</a> <a data-t="223">night, time to meet.</a> <a data-t="223">But you don't have to be that pushy.</a> <a data-t="223">Rather, the point of the facilitator is that</a> <a data-t="230">every week, there is an agenda.</a> <a data-t="230">And if no one is able to give that presentation for that week,</a> <a data-t="236">then the onus is on the presenter to take that upon themselves, or else the book might kind of</a> <a data-t="241">like falter to the wayside.</a></p>

<p><a data-t="251">The other component is collaborative infrastructure.</a> <a data-t="251">And for our case,</a> <a data-t="251">we use GitHub.</a> <a data-t="257">And this image, as well as the text before this, are links.</a> <a data-t="257">So feel free to</a> <a data-t="259">go out to our GitHub.</a> <a data-t="265">But our GitHub is three folders.</a> <a data-t="265">The first is our presentations.</a> <a data-t="266">So</a> <a data-t="266">the actual markdown presentations from each week.</a> <a data-t="274">The second folder contains a </a><a href="/software/bookdown/">bookdown</a><a data-t="274"> of all the</a> <a data-t="274">questions and answers collated from the Slack channel, so that you don't have to sift through</a> <a data-t="279">that.</a> <a data-t="287">And the last folder is a data folder.</a> <a data-t="287">And in this folder, we kind of took that concept of</a> <a data-t="287">applications and took a dataset and drilled down into it, where each chapter, we applied that to</a> <a data-t="294">this dataset of our own.</a></p>

<h3>Side dishes: unexpected learnings</h3>

<p><a data-t="304">Now, to talk about side dishes.</a> <a data-t="304">So when we started this book club,</a> <a data-t="310">at the outset, the goal was essentially learn the contents of the chapters.</a> <a data-t="310">But we ended up</a> <a data-t="310">learning all of these external things, as well.</a> <a data-t="318">So I personally onboarded people onto how to</a> <a data-t="318">use GitHub in a collaborative sense and collaborate on a single project.</a> <a data-t="324">We also</a> <a data-t="324">learned a lot about our markdown, HTML and CSS.</a> <a data-t="331">And lastly, and maybe even most importantly,</a> <a data-t="331">we were still leveling up on our presentation skills in these times where we're all isolated</a> <a data-t="337">and alone.</a></p>
