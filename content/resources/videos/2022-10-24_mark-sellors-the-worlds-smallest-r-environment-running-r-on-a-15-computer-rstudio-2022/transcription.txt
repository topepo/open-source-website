<h3>What is a Raspberry Pi?</h3>

<p><a data-t="57">So what is a Raspberry Pi?</a> <a data-t="64">The Raspberry Pi computer is a family of computers that was</a> <a data-t="64">first launched in 2012, so about 10 years old now.</a> <a data-t="73">They're created by an organisation</a> <a data-t="73">here in the UK called the Raspberry Pi Foundation.</a> <a data-t="81">The foundation's aim is really to improve</a> <a data-t="81">outcomes for folks through computer science and education and things like that.</a> <a data-t="86">And the</a> <a data-t="86">way that they do that is they create a very, very low cost computer.</a> <a data-t="93">They started out with</a> <a data-t="93">a computer that cost $35.</a> <a data-t="98">It's about the size of a deck of cards.</a> <a data-t="98">And then there've been</a> <a data-t="98">multiple iterations of that over the years.</a> <a data-t="105">And then at some point they released a smaller</a> <a data-t="105">version of that computer, which only cost $15.</a> <a data-t="112">So now there are probably three main</a> <a data-t="112">members of the Raspberry Pi family.</a> <a data-t="118">We'll take a look at those in a little more detail.</a></p>

<p><a data-t="118">The Raspberry Pi 4 Model B is the most common of the Raspberry Pi family at the moment.</a> <a data-t="124">It's the newest kind of standard size version of the computer.</a> <a data-t="132">This particular one is in</a> <a data-t="132">a nice rainbow case, but the computer itself, you can see how small it is.</a> <a data-t="137">It's probably</a> <a data-t="137">roughly the size of a deck of cards, something like that.</a> <a data-t="145">They start at $35.</a> <a data-t="145">That's the 1GB</a> <a data-t="145">model.</a> <a data-t="154">They go all the way up to 8GB of RAM.</a> <a data-t="154">It's a 4-core CPU at 1.5GHz.</a> <a data-t="154">There's an SD</a> <a data-t="154">card slot underneath for storage and all the things you would expect from a modern</a> <a data-t="159">computer.</a> <a data-t="159">So USB 3.0, Wi-Fi, Bluetooth, HDMI out, that kind of thing.</a></p>

<p><a data-t="167">Next we have the Raspberry Pi Zero 2W.</a> <a data-t="167">This is the second iteration of the Zero size computer.</a> <a data-t="176">You can see that that one is just over an inch wide, a little larger than a stick of</a> <a data-t="182">gum, I guess.</a> <a data-t="191">These are the $15 computers.</a> <a data-t="191">So it has a 4-core CPU at 1GHz, 512MB of RAM,</a> <a data-t="191">Wi-Fi, Bluetooth, HDMI out, and again, the SD card for the operating system.</a></p>

<p><a data-t="201">Then finally, we have the largest of the Raspberry Pi computers, but that's only because it's</a> <a data-t="207">built inside this keyboard here.</a> <a data-t="207">So you can see the keyboard has similar slots to the</a> <a data-t="213">Raspberry Pi 4.</a> <a data-t="213">It's basically a Raspberry Pi 4 inside a keyboard case.</a></p>

<p><a data-t="221">It's important to remember that the Raspberry Pi is just a computer.</a> <a data-t="221">It's a small Linux</a> <a data-t="221">computer, and as such, works very similarly to most other computers, particularly computers</a> <a data-t="234">using the Linux operating system.</a></p>

<h3>The R4Pi project</h3>

<p><a data-t="239">So what about the R4Pi project?</a> <a data-t="239">What is it, and what does it do?</a></p>

<p><a data-t="243">So the R4Pi project provides pre-built binaries of the R language, first of all, but also</a> <a data-t="252">provides a CRAN-like repository where we make available pre-compiled versions of CRAN packages.</a> <a data-t="270">Now, we don't do all of CRAN, maybe one day.</a> <a data-t="270">Right now, we're just doing a subset of CRAN.</a> <a data-t="275">So we're building somewhere in the region of 250, maybe 300 packages, something like that</a> <a data-t="275">at the moment, but we're adding more all the time.</a></p>

<p><a data-t="279">The way that this benefits users is that if you're unaware, if you're a Linux user and</a> <a data-t="288">you use CRAN, CRAN gives you the source code rather than a pre-compiled binary like they</a> <a data-t="293">do for Windows and Mac.</a> <a data-t="300">This is great because R knows how to build them and things like</a> <a data-t="300">that, but it can be very time-consuming.</a> <a data-t="304">There are often dependencies which aren't necessarily</a> <a data-t="304">met in your build environment, and so the R4Pi project builds those packages for you</a> <a data-t="311">and then makes them available.</a> <a data-t="317">In this regard, it's very similar to </a><a href="/software/rstudio/">RStudio</a><a data-t="317">'s package manager,</a> <a data-t="317">but obviously on a much smaller scale.</a></p>

<p><a data-t="321">With those packages available, it becomes much easier for R users on the Raspberry Pi</a> <a data-t="326">to install those packages.</a> <a data-t="332">The installation is much quicker and there's no build that</a> <a data-t="332">needs to occur.</a> <a data-t="337">And from there, people can get started much more quickly with R itself</a> <a data-t="337">without having to worry about all of this build stuff.</a></p>

<h3>Edge computing and the possibilities of small devices</h3>

<p><a data-t="814">One of the things that I'm most excited about about running R on a Raspberry Pi</a> <a data-t="819">is because it's a very small computer, you can run them in places where you wouldn't</a> <a data-t="825">normally run a $1,000, $2,000 computer.</a> <a data-t="834">They can be powered by batteries in a lot of cases,</a> <a data-t="834">especially the smaller ones.</a> <a data-t="840">You can run those from a USB battery bank.</a> <a data-t="842">But you can imagine</a> <a data-t="854">ordinarily with data science, people give us data.</a> <a data-t="854">So the data is collected somewhere.</a> <a data-t="848">It's bundled together, often not very well, and then handed over to the data scientists</a> <a data-t="853">for analysis.</a> <a data-t="859">But what we can do when we have a small computer like the Raspberry Pi,</a> <a data-t="859">is we can put that computer where the data is collected.</a> <a data-t="865">So that could be the data collection</a> <a data-t="865">device as well as the place where the analysis occurs.</a> <a data-t="871">This kind of concept in computing is</a> <a data-t="871">often referred to as edge computing because you can take the centralized resources that are often</a> <a data-t="879">used for this sort of work and you can move them to the edge of the network.</a> <a data-t="884">This means that you</a> <a data-t="884">can run projects in a lot more interesting places than you might otherwise be able to.</a></p>

<p><a data-t="892">Because of the low power requirements of the Raspberry Pi, you can run them in a car while</a> <a data-t="897">you're out power sending.</a> <a data-t="903">I can imagine people running them while they're, you know, on a</a> <a data-t="903">building site or in a digger.</a> <a data-t="911">Maybe even a farmer looking after cows in a field,</a> <a data-t="913">in a forest, on a boat.</a> <a data-t="921">The possibilities really are endless with such a small low power device.</a></p>

<p><a data-t="923">There are many different ways that you could run the types of projects that we've talked about</a> <a data-t="929">here today.</a> <a data-t="934">You don't need a Raspberry Pi computer to do any of this stuff.</a> <a data-t="935">But for me, the kind of low cost, the kind of ease of entry into the ecosystem</a> <a data-t="943">is a really big boon for a certain class of users.</a> <a data-t="948">Whether they're people who are just, you</a> <a data-t="948">know, coming to R for the first time, just want to learn.</a> <a data-t="953">People want to learn Linux so that they can</a> <a data-t="953">use R on Linux.</a> <a data-t="960">Because obviously that's more or less a de facto operating system for scientific</a> <a data-t="960">and technical computing at this point.</a> <a data-t="968">Or for any other reason.</a> <a data-t="968">Just having something that's very</a> <a data-t="968">low cost and very accessible for folks is a really nice way to help people get into the R ecosystem.</a></p>
