<h3>Inclusive pedagogy: the four A's</h3>

<p><a data-t="340">So we want to work towards this growth mindset using inclusive pedagogy.</a> <a data-t="344">So not focusing, we do focus on what we teach, that is a big thing, and I'm not talking about</a> <a data-t="350">that right now, but how we teach.</a> <a data-t="354">And so that's what I'm going to talk about now.</a> <a data-t="356">And so we're thinking about it in the context, and for the people that are in the room that</a> <a data-t="359">are saying, I've never actually heard us say it this way, that is true.</a> <a data-t="363">So we're thinking about taking all of the things that we do every day and stepping back</a> <a data-t="367">and saying, what is it that this represents?</a> <a data-t="370">It's these four A's, accessible, approachable, aligned, and active.</a></p>

<p><a data-t="414">That it's welcoming, so that when they are there, that it is accessible to them because</a> <a data-t="419">of the environment that's created.</a> <a data-t="421">So in all of our workshops, we have a code of conduct, and we have a reporting structure.</a> <a data-t="426">Those are both important, not just having a code of conduct.</a> <a data-t="430">And we start with that, I appreciate in this meeting, right, that they started with a code</a> <a data-t="434">of conduct, because it sets the stage as saying that this is a place that is supposed to be</a> <a data-t="439">accessible and welcoming to everybody.</a></p>

<p><a data-t="468">Approachable.</a> <a data-t="469">So, great, this is accessible, I can go to this thing, but do I want to go?</a> <a data-t="475">Is this for me?</a> <a data-t="476">And so that starts at the very beginning, in advertising and recruitment for the workshop.</a> <a data-t="481">If this is for beginners, say this is for people with little to no prior experience.</a> <a data-t="487">In this domain, map out, essentially, what your persona is of this person who can come,</a> <a data-t="493">and tell these people who it's for and why they're welcome.</a> <a data-t="497">Invite them to participate.</a></p>
