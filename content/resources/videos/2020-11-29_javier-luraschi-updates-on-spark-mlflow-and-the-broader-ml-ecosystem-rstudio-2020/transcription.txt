<h3>Delta Lake</h3>

<p><a data-t="268">So what is Delta Lake?</a> <a data-t="270">Delta Lake is a new open-source project</a> <a data-t="272">which happens to be hosted with the Linux Foundation,</a> <a data-t="275">and what it adds, it gives Spark the ability to have ACID transactions,</a> <a data-t="281">which is basically database-like properties</a> <a data-t="283">to use on top of your data lake.</a></p>

<p><a data-t="286">Now, this doesn't mean that you should use...</a> <a data-t="289">This doesn't mean that you should replace your databases with Spark.</a> <a data-t="292">That's not at all the goal of this project.</a> <a data-t="294">But what it really allows you to do is to mix real-time data</a> <a data-t="297">with batch processing data on the same table.</a> <a data-t="299">So in the past </a><a href="/software/rstudio/">RStudio</a><a data-t="299"> conference,</a> <a data-t="301">I had to do this demo where we used different types of functions,</a> <a data-t="304">and to load streaming data against batch data.</a> <a data-t="307">And basically, you can replace all that with Delta Lake</a> <a data-t="310">because you don't need to worry about where the data is coming from.</a> <a data-t="313">It just goes to the same source.</a> <a data-t="315">And it also enables a few nice features,</a> <a data-t="317">like being able to find and use previous versions of your datasets,</a> <a data-t="321">which we'll see in a second.</a></p>

<p><a data-t="323">So how do you use Delta Lake from R?</a> <a data-t="326">You need to load the sparklyr package,</a> <a data-t="328">and then you connect as usual.</a> <a data-t="330">But there's a new parameter called packages,</a> <a data-t="332">so you want to specify packages equals Delta.</a> <a data-t="334">And then you basically use the functions to read and write data.</a> <a data-t="337">You say, Spark write Delta, and you write a data frame.</a> <a data-t="340">Here we have huge data frames with three rows,</a> <a data-t="343">and we're replacing that huge data frame with another data frame with one row.</a> <a data-t="348">And basically, then if you ask Delta Lake for that particular dataset,</a> <a data-t="352">you just say, Spark read Delta,</a> <a data-t="354">and you get that dataset right with one row.</a> <a data-t="356">But what is interesting is you can also tell Delta,</a> <a data-t="360">hey, give me the previous version that I just overwrote.</a> <a data-t="363">And what that would give you back is the tree records that were initially there.</a></p>
