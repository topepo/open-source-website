<p><a data-t="248">The first one I'm going to add is Jupyter, so uv pip install Jupyter.</a> <a data-t="253">And Jupyter is required by Quarto to be able to execute Python code cells.</a> <a data-t="258">So if you've got Python code cells, you're going to need the Jupyter package.</a> <a data-t="263">And then the other thing I'm going to install is Plot9.</a> <a data-t="265">This is now my analysis-specific.</a> <a data-t="268">I'm going to make some plots with Plot9, so I'm going to install that package as well.</a></p>

<p><a data-t="274">Now we can go ahead and add a code cell.</a> <a data-t="277">And again, there's lots of ways to do this.</a> <a data-t="279">There's an Insert Code Cell button up in the toolbar.</a> <a data-t="282">There's a shortcut for that.</a> <a data-t="284">There's also a command for that called Quarto Insert Code Cell.</a> <a data-t="288">I'm going to click that button, and then a dropdown comes up for which language this</a> <a data-t="294">code cell will be in, and I want a Python code cell.</a> <a data-t="296">So I just hit Enter for that.</a></p>

<p><a data-t="299">So now I can write my Python code.</a> <a data-t="300">The first thing I want to do is import some packages.</a> <a data-t="304">So from Plot9, I'm going to import all the functions, and from Plot9 Data, I'm going</a> <a data-t="314">to import penguins.</a></p>

<p><a data-t="317">Now I'm just writing this code in a code cell.</a> <a data-t="318">I haven't done anything to execute it.</a> <a data-t="321">But the nice thing here is it's really easy to take this code and send it down to my console.</a> <a data-t="325">Again, lots of ways to do it.</a> <a data-t="328">There's this Run Cell button that'll appear above any code cell.</a> <a data-t="331">If I click that, it sends it off to my console.</a></p>

<p><a data-t="619">Typed is an alternative to LaTeX.</a> <a data-t="621">It creates PDFs.</a> <a data-t="621">One of the great things about it is it's bundled in Quarto.</a> <a data-t="624">So you can create PDFs with Quarto without any other installations.</a> <a data-t="629">And up pops a preview of it that's now a PDF document.</a> <a data-t="634">And I'm viewing that in the viewer pane.</a></p>

<p><a data-t="672">And I'm going to tell it I want to publish this document, the standalone document.</a> <a data-t="676">So penguins.qmd.</a> <a data-t="678">And Quarto publish will pop up a list of places you can publish.</a> <a data-t="682">I'm going to choose Quarto pub because that's just a really easy, quick way to get started.</a> <a data-t="687">I already have an account on Quarto pub with my email address.</a> <a data-t="693">And it's going to ask me what I want to call this document.</a> <a data-t="696">I'll just call it penguins.</a> <a data-t="698">And Quarto will handle the rest.</a> <a data-t="699">It's going to rerender that document, make sure it's up to date.</a> <a data-t="702">It's going to send all those files up to Quarto pub.</a></p>
