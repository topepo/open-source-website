<h3>Installing with pip in <a href="/software/positron/">Positron</a></h3>

<p><a data-t="25">First, we'll head over to the terminal tab in Positron.</a> <a data-t="29">We'll use Pip, Python's go-to package manager, to handle the installation.</a> <a data-t="34">In the terminal, you'll simply type pip install PyTuesday.</a> <a data-t="40">As soon as you hit enter, Pip gets to work, automatically downloading the library and any other dependencies it needs.</a></p>

<p><a data-t="48">Think of it like this.</a> <a data-t="50">Just as you download a book from an online store, we're downloading the PyTuesday library from a vast online collection of Python libraries called the Python Package Index, or PyPI.</a></p>

<h3>Managing installed packages</h3>

<p><a data-t="65">If you ever want to see which packages are currently installed in your environment, just type pip freeze in the terminal.</a> <a data-t="73">This command lists every library along with its version number.</a> <a data-t="77">This output is incredibly useful for reproducing your project's environment, which is why you'll often see the output of this command saved to a file called requirements.txt.</a></p>
