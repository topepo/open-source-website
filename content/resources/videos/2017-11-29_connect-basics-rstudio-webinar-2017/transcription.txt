<h3>What is RStudio Connect</h3>

<p><a data-t="72">So RStudio Connect is an enterprise publishing and management platform.</a> <a data-t="76">It can handle static and dynamic content, basically anything that you can create with</a> <a data-t="80">R, including documents, presentations, dashboards, and </a><a href="/software/shiny-r/">Shiny</a><a data-t="80"> applications.</a> <a data-t="85">And so one of the nice things about this is that it has push-button publishing from</a> <a data-t="89">the RStudio IDE.</a></p>

<p><a data-t="90">And so if you're using a recent version of the IDE, you'll see this little publish button</a> <a data-t="94">that you see here at the top right.</a> <a data-t="96">You should be able to click that and click Publish to RStudio Connect, and if you register</a> <a data-t="100">an account on an RStudio Connect server, you'll be able to seamlessly publish documents and</a> <a data-t="104">R Markdown documents and Shiny applications and anything else that you're producing in</a> <a data-t="109">the IDE into Connect.</a></p>

<p><a data-t="111">It is an on-premises enterprise product, so this is not an open-source product, but it</a> <a data-t="115">is something that you can install behind your firewall, and it's not a hosted environment.</a> <a data-t="122">And lastly, it obviously is meant for sharing data science artifacts within your organization.</a></p>
