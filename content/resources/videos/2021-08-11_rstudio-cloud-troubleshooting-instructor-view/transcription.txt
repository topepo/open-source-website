<p><a data-t="0">While we hope that you have a seamless </a><a href="/software/rstudio/">RStudio</a><a data-t="0"> Cloud experience, occasionally things might</a> <a data-t="7">not work as intended.</a> <a data-t="9">Sometimes when we're working on a project, it may fail to load or become unresponsive.</a> <a data-t="15">When this happens, we can relaunch the project by clicking on the three dots inside a circle</a> <a data-t="20">to the left of the settings icon at the top of our project.</a> <a data-t="23">Choosing the third option from this menu will relaunch our project.</a></p>

<h3>Checking system status and getting help</h3>

<p><a data-t="29">If this doesn't solve your issue, you can check to see if there are any broader issues</a> <a data-t="33">with RStudio Cloud.</a> <a data-t="35">To do this, expand the menu on the left to expose the current system status option in</a> <a data-t="41">the help menu.</a> <a data-t="42">Clicking this will tell us if RStudio Cloud is running as intended.</a> <a data-t="47">If RStudio Cloud is operational and you are still running into issues, you may choose</a> <a data-t="52">to post on our community forum or reach out to RStudio's technical support team.</a></p>
