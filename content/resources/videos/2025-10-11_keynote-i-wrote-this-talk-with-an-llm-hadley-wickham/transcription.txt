<h3>Tool calling and agents</h3>

<p><a data-t="996">The next thing I want to talk about is tool calling, and I want to illustrate that with</a> <a data-t="1000">a problem.</a> <a data-t="1001">Another thing that LLMs are bad at.</a> <a data-t="1003">If you ask an LLM what day it is today, best-case scenario, it's going to say, I don't know.</a> <a data-t="1009">If you ask this of chatGBT, it actually just makes up a date, which is not very reassuring.</a> <a data-t="1016">And LLMs like can't do anything, right, they're stuck in a box.</a> <a data-t="1020">You can't say, hey LLM, like delete the CSV files on my directory.</a> <a data-t="1025">And we can get around both of these limitations by using these things called tools.</a></p>
