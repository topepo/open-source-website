<p><a data-t="31">I'm going to authorize access, and then I'm going to be able to go back into the IDE to</a> <a data-t="31">continue with the deployment.</a> <a data-t="37">Once here, I can pick which account that I want,</a> <a data-t="38">and I can use another RSConnect command, deploy app, to put this onto Connect Cloud.</a> <a data-t="44">It's going</a> <a data-t="44">to confirm which account that I want.</a> <a data-t="49">So we'll go to Connect Cloud, we'll do our deployment,</a> <a data-t="49">we'll get all of the dependencies that are needed from our local environment,</a> <a data-t="53">and now we're going to be able to see this show up on Connect Cloud.</a> <a data-t="59">And here's the final document.</a></p>

<h3>Fine-grained compute and runtime settings</h3>

<p><a data-t="61">Number two, set fine-grained compute and runtime settings.</a> <a data-t="61">So let's go into a </a><a href="/software/shiny-r/">Shiny</a><a data-t="61"> application.</a> <a data-t="69">This is an A-B test calculator.</a> <a data-t="73">If I increase the number of conversions on the B side,</a> <a data-t="73">we're going to see all of a sudden we have statistical significance.</a> <a data-t="77">But I can also go</a> <a data-t="77">into edit settings, and we're going to see a bunch of new tabs, one for compute.</a></p>
