<h3>About the science and the package</h3>

<p><a data-t="162">I feel like I should tell you a little</a> <a data-t="162">bit about the package.</a> <a data-t="167">In doing so, I really need to tell you a little bit about the science.</a> <a data-t="167">I studied genetics of complex traits, mostly in mice.</a> <a data-t="173">Mice have a big advantage for genetics</a> <a data-t="173">in that you can make inbred lines.</a> <a data-t="179">You can take a pair of mice and mate them and then</a> <a data-t="179">do repeated mating between siblings for 20 generations or 200 generations.</a> <a data-t="186">What you get</a> <a data-t="186">at the end is a strain of mice that are all genetically identical.</a> <a data-t="192">If you can figure out</a> <a data-t="192">which one is a male and which one is a female, cross them, the offspring will all be genetically</a> <a data-t="197">identical again.</a></p>

<p><a data-t="204">Our interest in the genetics in mice is really</a> <a data-t="214">focused on understanding the genetics of human disease.</a> <a data-t="219">We're not trying to cure disease</a> <a data-t="219">in mice.</a> <a data-t="228">We're trying to have a better understanding of the underlying biology of human disease.</a></p>

<p><a data-t="228">The basic experiment that's done, it would be an intercross where you take two strains</a> <a data-t="234">of mice that differ in some trait.</a> <a data-t="241">One strain has high blood pressure.</a> <a data-t="241">The other strain</a> <a data-t="241">has low blood pressure.</a> <a data-t="244">You're trying to find what genes are contributing to that difference</a> <a data-t="244">in blood pressure.</a> <a data-t="251">You do a cross that's identical to what Mendel did with his peas to mix up</a> <a data-t="259">the high blood pressure chromosomes and the low blood pressure chromosomes.</a> <a data-t="260">Then what</a> <a data-t="260">you're going to do is get a bunch of offspring, measure blood pressure in all those mice,</a> <a data-t="268">which really for that phenotype would be the hardest part of this study.</a> <a data-t="275">Measure the blood</a> <a data-t="275">pressure of all those offspring and measure genotype along each chromosome.</a> <a data-t="280">Whether a</a> <a data-t="280">mouse is blue-blue, pink-pink, or blue-pink at a given position.</a> <a data-t="286">Then we're going to scan</a> <a data-t="286">across the genome and look for reasons of the genome where there's an association between</a> <a data-t="292">what genotype you have, whether you were blue-blue, blue-pink, or pink-pink, whether that's associated</a> <a data-t="297">with the outcome blood pressure.</a></p>

<p><a data-t="305">The results of this sort of analysis is a plot across</a> <a data-t="305">the mouse genome on each chromosome, the curves for a test statistic that's indicating when</a> <a data-t="314">this thing is high, you are saying there's evidence for an association between genotype</a> <a data-t="319">at that location and blood pressure.</a> <a data-t="327">Those peaks we call QTL or quantitative trait loci,</a> <a data-t="327">a region of the genome that is affecting a quantitative trait blood pressure.</a> <a data-t="337">Effectively</a> <a data-t="337">what you do is at every position in the genome you do an analysis of variance to see is there</a> <a data-t="342">a difference in the average trait across the three genotypes.</a> <a data-t="349">The goal of my package RQTL</a> <a data-t="349">is called QTL because we call these sites we're looking for QTL, the genetic loci that</a> <a data-t="359">affect this quantitative trait.</a> <a data-t="365">It does a bunch of other things.</a></p>
