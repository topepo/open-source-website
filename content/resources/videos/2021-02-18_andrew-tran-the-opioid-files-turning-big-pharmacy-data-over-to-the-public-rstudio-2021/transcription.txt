<h3>The ARCOS data set</h3>

<p><a data-t="89">So we've seen the effects, but we've never really been able to see the causes before,</a> <a data-t="94">not with a lot of specificity.</a> <a data-t="96">But the thing is, pharmaceutical companies and the Drug Enforcement Administration have</a> <a data-t="103">been able to see this for the longest time, because every single pill from manufacturer</a> <a data-t="108">to distributor to pharmacy is tracked in a data set, and is called the Automated Reports</a> <a data-t="119">and Consolidated Ords System.</a></p>

<p><a data-t="124">And the Drug Enforcement Administration is also under the Department of Justice.</a> <a data-t="128">And it's tracked because prescription pain pills are scheduled to drug, which means it</a> <a data-t="135">has a high potential for abuse, with use potentially leading to severe psychological</a> <a data-t="140">and physical dependence.</a> <a data-t="142">And we call it ARCOS for short.</a></p>

<p><a data-t="148">And so pharmaceutical companies and the government have known the scope of this data for the</a> <a data-t="153">longest time, the reach of the spread of these pills.</a> <a data-t="158">And journalists and researchers have tried to get this data set for the longest time</a> <a data-t="162">as well.</a> <a data-t="162">But we've only been able to get it at the aggregate level, maybe by state, maybe by</a> <a data-t="166">county, maybe by zip code, maybe by overall drug.</a> <a data-t="171">We've never been able to get it at the raw data set level, where we could really dig</a> <a data-t="175">into the nuances of the data.</a> <a data-t="177">Every time we've made a request for it, it's been denied.</a></p>

<p><a data-t="182">That is until recently.</a></p>

<p><a data-t="188">So for context, the most powerful people in the pharmaceutical America didn't want</a> <a data-t="192">it released.</a> <a data-t="193">And a lot of the most powerful people in the Department of Justice didn't want this data</a> <a data-t="196">released.</a> <a data-t="197">However, when 2,000 communities across the US sued dozens of big drug companies over</a> <a data-t="203">the destruction that painkillers were causing across the country, a federal judge ordered</a> <a data-t="209">that the ARCOS data set be made available to those plaintiffs for examination.</a> <a data-t="215">However, they sealed it away from the public and from the media.</a></p>

<p><a data-t="221">And so the Washington Post, as well as the Charleston Gazette-Mail sued for access to</a> <a data-t="226">that data.</a> <a data-t="228">And so we appealed the decision.</a> <a data-t="231">And a group of appellate judges ruled that the protective order on the ARCOS data should</a> <a data-t="235">be amended.</a> <a data-t="237">In effect, they released data from 2006 to 2012, and eventually 2014.</a> <a data-t="246">And so we got our hands on the data the day it was released.</a></p>

<h3>Analyzing the data</h3>

<p><a data-t="250">And what we found out was that the data set was huge.</a> <a data-t="254">It had about 42 columns, which isn't too bad, but it had literally hundreds of millions</a> <a data-t="259">of rows of data for every pain pill order sold between 2006 and eventually 2014.</a> <a data-t="266">And so the team of us, there were about three data reporters, including myself.</a> <a data-t="270">We used a mix of different statistical analyzing languages, like Python and R.</a> <a data-t="276">And so in order to maintain great communication between the two, we used data structures that</a> <a data-t="282">could operate between the two languages, such as Parquet and Feather.</a> <a data-t="286">And because the data sets were so huge, we used packages like doParallel in order to</a> <a data-t="293">handle all that.</a></p>

<p><a data-t="297">And in order to exchange our analysis methodologies and reports, we wrote up everything in Jupyter</a> <a data-t="306">Notebooks or R Markdown.</a> <a data-t="309">I also created a </a><a href="/software/shiny-r/">Shiny</a><a data-t="309"> app so that we could investigate and dig into pharmacies that looked</a> <a data-t="317">suspicious, because any way you might have sliced it would raise a different set of pharmacies</a> <a data-t="322">that might have set off some suspicious orders.</a></p>
