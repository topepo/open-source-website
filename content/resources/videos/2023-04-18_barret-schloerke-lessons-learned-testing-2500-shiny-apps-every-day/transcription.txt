<h3>GitHub Actions and automation</h3>

<p><a data-t="745">But about this time, though, something came around called GitHub Actions, and GitHub Actions is a continuous integration platform that can automate your workflows within world-class CI, CD.</a> <a data-t="763">That's their tagline.</a> <a data-t="764">And they also said to build, test, and deploy your code right from GitHub.</a> <a data-t="769">And I think this is huge, because it's not going to be Travis CI or Circle, which while they worked, it was somewhere else, and it's not integrated in with your PR and your code and everything else.</a> <a data-t="790">And so it was really, really intriguing when this came around.</a></p>

<p><a data-t="796">They had some growing pains, but it turned out in their V2 or what we use today to be pretty stable and really intriguing.</a> <a data-t="806">The better part is that the pricing is free for public repos.</a> <a data-t="811">It's not free for private, but everything we do within the Shiny team is public.</a> <a data-t="817">And so, sweet, it's going to be free for us.</a></p>

<p><a data-t="823">There's a repo called RLib Actions, and it's a collection of GitHub Actions for the R language.</a> <a data-t="831">And it's currently maintained by Gabor and was originally created by Jim Hester.</a> <a data-t="836">And it has many actions for you to use, kind of like Lego blocks.</a> <a data-t="840">You can use them or not use them, depending on your needs.</a> <a data-t="844">And you can do things like setting up Pandoc, setting up R, setting up R dependencies for your package, and then also checking your R package.</a> <a data-t="852">And these actions, you can, as I said earlier, you can add in and expand on them, you can adjust them, but also you can do it in a way that you don't know what operating system you're on.</a> <a data-t="870">If you've ever done testing with some platform, you usually have a very, very specific codebase for Ubuntu.</a> <a data-t="880">You have a very specific codebase for Mac, and you have a different codebase completely for Windows.</a> <a data-t="887">And here you just say, hey, set up Pandoc.</a> <a data-t="890">Thank you.</a> <a data-t="891">Set up R for me.</a> <a data-t="892">Thank you.</a> <a data-t="893">And that is really, really powerful that you can just put those in and all that work is hidden underneath.</a></p>
