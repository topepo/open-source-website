<h3>What are microservices?</h3>

<p><a data-t="492">So it has led us to microservices. If we take a look at those different challenges and those</a> <a data-t="496">balancing needs, we want to start talking about microservices. <h3>Challenges to getting started with microservices</h3>

<p><a data-t="1069">So there are four pretty overarching challenges with getting started with microservices, and</a> <a data-t="1076">the first one is getting started with microservices can be very daunting.</a> <a data-t="1080">There's a lot of material out there.</a> <a data-t="1083">Microservices are defined very similarly, but slightly different depending on which source you</a> <a data-t="1089">look at. <p><a data-t="1120">On top of that, because these services can be used with different user interfaces, it</a> <a data-t="1128">becomes a lot more important to define what that interface is, because if you do have to</a> <a data-t="1134">change it in the future, you have to go back and you have to test all the older versions or</a> <a data-t="1138">handle version control very carefully.</a> <a data-t="1141">So getting started is quite a challenge and we'll talk to that a bit here.</a></p>
