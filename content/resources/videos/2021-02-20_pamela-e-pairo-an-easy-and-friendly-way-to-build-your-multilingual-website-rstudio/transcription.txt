<p><a data-t="44">And finally, the personal website is a platform to connect with the world.</a> <a data-t="50">For those of us</a> <a data-t="50">who aren't native English speakers, it is fundamental to have a multilingual website</a> <a data-t="55">in English to disseminate our work to the global community and in our native language</a> <a data-t="61">to make our website content more inclusive and accessible for everyone.</a></p>

<h3>Getting started with blogdown and Hugo</h3>

<p><a data-t="67">Let's get started.</a> <a data-t="67">We begin creating a new repo in our GitHub account.</a> <a data-t="75">Then, in </a><a href="/software/rstudio/">RStudio</a><a data-t="75">, create a new project</a> <a data-t="75">with Contribution, Undownload, blogdown, and Hugo.</a> <a data-t="80">You can look at all the themes and</a> <a data-t="80">select one on the Hugo website.</a> <a data-t="87">After that, you will have the default version of the site</a> <a data-t="87">in the viewer panel, so it's time to customize.</a></p>

<p><a data-t="93">The main files that are part of the website</a> <a data-t="93">are shown here.</a> <a data-t="99">In </a><a href="/software/config/">config</a><a data-t="99">.tom is where it's located the global settings for the site and</a> <a data-t="99">in config.default folder are the params.toms where you can change the color theme, the</a> <a data-t="106">font size of your website, and add your contact details.</a> <a data-t="114">To change what appears in the navigation</a> <a data-t="114">menu bar, it's necessary to edit the menu.tom file.</a> <a data-t="119">And in language.tom is where you can</a> <a data-t="119">change the language setting.</a></p>

<h3>Making your website multilingual</h3>

<p><a data-t="126">And how to make your personal website multilingual?</a> <a data-t="126">I'm going</a> <a data-t="126">to show you an example of the bilingual website, English and Spanish.</a> <a data-t="134">In the language.tom,</a> <a data-t="134">you have to uncomment the lines for a second language to configure your website.</a> <a data-t="141">Here we</a> <a data-t="141">have the information about the English part, for example, the language code, where it's</a> <a data-t="147">located the content, and the title.</a> <a data-t="153">Note that the abbreviations of the languages are predefined,</a> <a data-t="153">so if you want a Portuguese website, for example, you need to check which is the correct abbreviation.</a></p>

<p><a data-t="160">Then, in each language, we'll need their own menu.tom file and a subfolder within the content</a> <a data-t="169">folder.</a> <a data-t="177">Finally, choose the default language that the visitors will see when visit your page,</a> <a data-t="177">and for making visible that the website is multilingual, check in params.tom that show</a> <a data-t="184">is true.</a></p>

<h3>Deploying with Netlify</h3>

<p><a data-t="191">When you are done with your edits and you are happy with your website, it's time to</a> <a data-t="191">put it online.</a> <a data-t="198">One way to do this is using Netlify.</a> <a data-t="198">One of the advantages of deploying</a> <a data-t="198">your website through Netlify is that it allows you to have a static site for free and easily.</a> <a data-t="204">Basically, it connects to the remote repository on GitHub to deploy.</a> <a data-t="210">Every change that it makes</a> <a data-t="210">on the website, it will be automatically updated after to push to GitHub.</a></p>

<p><a data-t="221">It's important that the</a> <a data-t="221">Hugo version that you are running locally match with the version that Netlify is going to use to</a> <a data-t="229">build your site, and how to do that?</a> <a data-t="238">Check your local version of Hugo and pick up the Netlify</a> <a data-t="238">file to your project root directory and specify the Hugo version.</a> <a data-t="248">Then, go to the Netlify webpage</a> <a data-t="248">and log in with GitHub to select the repo you've been working with blogdown.</a> <a data-t="257">You will see something</a> <a data-t="257">like this but with your information.</a> <a data-t="265">Go to show advanced and just in case, write the Hugo version,</a> <a data-t="265">and finally, you will have your website.</a></p>
