<h3>Introducing RStudio Connect</h3>

<p><a data-t="156">But we solved these problems with RStudio Connect.</a> <a data-t="160">If you're unfamiliar, RStudio Connect is a content hub and execution engine for your data products.</a> <a data-t="167">And so what that specifically means as an analyst is that your first experience is going to be clicking a little Publish button in the RStudio IDE, which I'll show you.</a> <a data-t="175">And that will do all the hard work of getting your content from your computer onto the remote machine.</a></p>

<p><a data-t="180">You can schedule and distribute our markdown content. And so now you get an easier way to be able to operationalize some of the content that you've produced.</a> <a data-t="189">You can handle the kind of scaling problems that we've been discussing about all morning around Shiny applications to be able to send more and more traffic towards Shiny or </a><a href="/software/plumber/">Plumber</a><a data-t="189"> APIs that you've deployed.</a> <a data-t="198">And one of the more important features, you can manage access controls.</a> <a data-t="201">And so you can actually go in and say, these users or these groups should be able to access this report, but nobody else is going to be able to see it.</a></p>

<p><a data-t="208">And an important caveat, this is an on-premises commercial application. So this is not one of our open source contributions, but this is something that you can purchase and run inside your firewall on-premises.</a> <a data-t="220">And so this will integrate with your directory of users and groups and all that kind of stuff.</a></p>

<p><a data-t="223">So a couple things that I'm excited to announce today that are available in the 1.7.0 release.</a> <a data-t="228">First of all, it's programmatic deployment. And so most people tend to reach for the push button deployment from the RStudio IDE.</a> <a data-t="235">But you also have the option now of saying, if you want to point your Jenkins or your Bamboo server at a Git repo and have it monitor that and automatically pull down changes and publish those, you now have that option with the programmatic deployment.</a></p>

<p><a data-t="246">We have customizable email, if you haven't seen this yet.
