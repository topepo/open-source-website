<h3>Why change was needed</h3>

<p><a data-t="157">So why do we need to change?</a> <a data-t="162">Maybe a third of you will recognise this thing.</a> <a data-t="162">This is a floppy disk that holds about one meg worth of data.</a> <a data-t="166">So historically, 10, 20 years ago when we were doing a farm trial,</a> <a data-t="172">we'd get a few observations per day.</a> <a data-t="175">So you could have a major trial where all the data would fit</a> <a data-t="179">on about six of these floppy disks.</a></p>

<p><a data-t="182">However, today we have wearable devices.</a> <a data-t="185">You can imagine Apple Watches for cows.</a> <a data-t="188">Devices like this are generating 60 observations a second</a> <a data-t="193">and you're putting this across hundreds and sometimes thousands of cows</a> <a data-t="199">and that's generating a metric crap tonne of data.</a></p>

<p><a data-t="204">And so if you put all the data for a three-month trial today onto floppy disks,</a> <a data-t="210">the tower of floppy disks would be taller than the Burj Khalifa,</a> <a data-t="213">like the tallest building in the world.</a></p>

<p><a data-t="217">Also at the same time, we had to move from linear science</a> <a data-t="223">in the sense of scientists would have an experiment,</a> <a data-t="227">they'd run the experiment, they'd throw some data at the statistician,</a> <a data-t="231">produce some plots and some tables, publish it, job done.</a> <a data-t="235">But when we're looking at this more complex data,</a> <a data-t="238">it becomes more iterative in the sense of looking at the data,</a> <a data-t="241">understanding what it's trying to tell us,</a> <a data-t="243">come up with some hypotheses, test those.</a> <a data-t="246">So a very different way of working.</a></p>

<p><a data-t="249">So these are not all the reasons,</a> <a data-t="251">but these were a couple of the obvious reasons that created</a> <a data-t="254">that burning platform for why we needed to do something different.</a></p>

<h3>Infrastructure</h3>

<p><a data-t="259">So we did three things.</a> <a data-t="262">The first thing was around infrastructure.</a> <a data-t="265">So if you've opened the R for Data Science book,</a> <a data-t="268">this diagram's probably quite familiar.</a> <a data-t="270">Anyone that's got R can get started and use this workflow.</a> <a data-t="274">But when you want to work at scale, you want to add a few things.</a></p>

<p><a data-t="278">So for us, we've got a data platform.</a> <a data-t="281">We chose Snowflake.</a> <a data-t="283">You might choose Databricks or something else.</a> <a data-t="285">For sharing code and versioning code, we use GitHub.</a> <a data-t="289">Again, there's other options there.</a> <a data-t="292">When the smoke is starting to pour out of your laptop,</a> <a data-t="295">it's time to move that compute to the cloud.</a> <a data-t="297">So we use Posit Workbench for that.</a> <a data-t="300">And then when we've created artifacts, tools, dashboards, et cetera,</a> <a data-t="303">that we want to share, then we push that through to Posit Connect.</a></p>

<p><a data-t="309">So what guided some of those choices?</a> <a data-t="311">Well, it needed to be fit for purpose,</a> <a data-t="313">and for us, that meant performance.</a> <a data-t="315">It needed to be scalable,</a> <a data-t="317">as we did more and more of this type of work,</a> <a data-t="320">and reproducible is just really good practice.</a> <a data-t="323">Also, we're starting from a low base,</a> <a data-t="326">so we want a relatively low cost to start.</a> <a data-t="329">So some open-source tools fit the bill there.</a> <a data-t="332">And you also need ease of management by your IT.</a> <a data-t="337">So that forms part of your total cost of ownership.</a></p>

<p><a data-t="344">You're going to need some confidence in the vendor,</a> <a data-t="348">but you also want to limit the technological lock-in</a> <a data-t="351">wherever possible, so you have options down the future.</a> <a data-t="354">You know, the landscape might change.</a> <a data-t="357">For us, support for R and Python,</a> <a data-t="360">we're primarily R now, but I could see there are some things</a> <a data-t="363">we might want to use Python for in the future,</a> <a data-t="366">and depending on where you're at and what your situation is,</a> <a data-t="369">you might make different choices.</a></p>

<p><a data-t="373">And so this involves spending money,</a> <a data-t="377">so you're going to need some allies from around the organization,</a> <a data-t="380">extra points if you get IT on board,</a> <a data-t="383">and then you're going to have to get the senior buy-in.</a> <a data-t="386">They're the people that are going to be writing the checks</a> <a data-t="388">for some of this stuff.</a></p>

<h3>Skills and capability</h3>

<p><a data-t="391">So the second of three things, the skills and capability.</a> <a data-t="395">So when we were starting down this journey,</a> <a data-t="398">we ran R for data science as an internal course.</a> <a data-t="403">We ran it as a cohort, so a bunch of people</a> <a data-t="406">all learning at the same time,</a> <a data-t="409">and the temptation is whenever you're doing something</a> <a data-t="412">in that setting that you try and just jam it all into,</a> <a data-t="415">like, say, three days.</a> <a data-t="417">We'll bring everyone in, smash them full of knowledge,</a> <a data-t="419">and then job done.</a></p>

<p><a data-t="421">But I took from the, what was the </a><a href="/software/rstudio/">RStudio</a><a data-t="421"> education piece</a> <a data-t="426">that they used to have around the space learning,</a> <a data-t="429">which is to say, you know, if you learn a bit this week,</a> <a data-t="432">then you've got time to digest it and use it,</a> <a data-t="435">and then more the next week, et cetera.</a> <a data-t="437">So that space learning as a concept, I think, is quite useful.</a></p>

<p><a data-t="444">You know, there was a few pros and cons to running it externally,</a> <a data-t="447">and we decided to go external for subsequent versions</a> <a data-t="451">and ran it through the university.</a> <a data-t="454">The things I like about doing it externally</a> <a data-t="456">is that you're getting some commitment</a> <a data-t="458">from both the person and the manager.</a> <a data-t="461">Even though it doesn't cost a lot,</a> <a data-t="463">you know, the fact that it actually costs something</a> <a data-t="465">means that that person and manager have to have a discussion</a> <a data-t="468">and they have to agree that this is a priority for this person,</a> <a data-t="472">and that means putting aside time to do it properly,</a> <a data-t="475">which is not necessarily what happened</a> <a data-t="478">when we had an internal course.</a> <a data-t="480">And if people got busy, then if it wasn't a priority,</a> <a data-t="483">it didn't happen.</a></p>

<p><a data-t="485">I also like the summative assessment,</a> <a data-t="487">so assessment at the end of the course,</a> <a data-t="489">which is to say, you know, people understand,</a> <a data-t="492">you know, most of these people have been to university,</a> <a data-t="494">so they understand that, you know,</a> <a data-t="496">there's going to be a test at the end</a> <a data-t="498">and they're going to have to have some minimum level of knowledge</a> <a data-t="501">in order to pass that and not be embarrassed in front of their peers.</a> <a data-t="505">So I like that.</a></p>

<p><a data-t="507">Working with the university, though,</a> <a data-t="509">you do have to get on the same page</a> <a data-t="512">because, you know, they might have different ideas</a> <a data-t="515">about how they want to teach it</a> <a data-t="517">and what's the place of AI tools in terms of teaching it.</a> <a data-t="522">From our perspective, like, we were quite pro-AI tools.</a> <a data-t="526">You know, we want the people to learn something,</a> <a data-t="528">but when they're in the workplace,</a> <a data-t="530">they're going to be using AI tools,</a> <a data-t="532">so we want them to learn how to work with AI tools</a> <a data-t="536">as part of that process.</a></p>
