<h3>How broom helps</h3>

<p><a data-t="173">Okay. And Broom is here to help.</a> <a data-t="173">So Broom is, uh, a package written by David Robinson and there's these kind of three key ideas.</a> <a data-t="178">The idea is that, uh, we can take these fits by fit.</a> <a data-t="183">What I mean is, um, you have some model and you like train a model and you give me a trained model back.</a> <a data-t="188">Okay. And we're going to say that there's kind of like three key components in this like Broom philosophy and we're going to, all of these components are going to be tidy tibbles.</a> <a data-t="196">And the reason I'm going to be tidy tibbles is because we know how to manipulate tibbles and so we can use tools that we already know and there isn't any kind of additional cognitive load in working with those things.</a></p>

<p><a data-t="209">So the first type of information about a fit is going to be like the components or the parameters.</a> <a data-t="213">So if you have a linear regression model, you could think about the regression coefficients.</a> <a data-t="216">So we're going to have a table that's going to have all the information about the regression coefficients.</a> <a data-t="222">Okay. <p><a data-t="233">Okay.
