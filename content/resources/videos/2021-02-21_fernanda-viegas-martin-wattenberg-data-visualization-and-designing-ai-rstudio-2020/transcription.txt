<p><a data-t="1318">It allows you to do things like click on a data point and say,</a> <a data-t="1323">for instance, let's go back to the loan scenario and say,</a> <a data-t="1327">oh, this person did not get a loan, OK?</a> <a data-t="1331">This person did not get a loan.</a> <a data-t="1338">Who is the nearest person to that one that did get a loan?</a> <a data-t="1342">And what is the delta there?</a> <a data-t="1344">So a little notion of counterfactual, if you will,</a> <a data-t="1348">without the causality.</a> <a data-t="1349">That's a separate conversation.</a> <a data-t="1351">But just this notion of starting to try to understand</a> <a data-t="1355">what's happening with these systems, how they are behaving,</a> <a data-t="1358">asking a bunch of alternative scenarios,</a> <a data-t="1361">trying a bunch of edge cases, and also</a> <a data-t="1365">playing with notions of fairness,</a> <a data-t="1368">of how different slices of your data set</a> <a data-t="1371">are doing vis-a-vis one another.</a></p>

<h3>TCAV: interpreting neural networks</h3>

<p><a data-t="2053">So we have this thing that's powerful.</a> <a data-t="2055">It requires a lot of analysis.</a> <a data-t="2057">But let's think about how we could</a> <a data-t="2059">use this idea of meaningful directions in other ways,</a> <a data-t="2062">now that we have it.</a> <a data-t="2064">So one way, it turns out, is to interpret</a> <a data-t="2067">what's going on inside of neural networks</a> <a data-t="2069">so that they're not, in fact, black boxes.</a> <a data-t="2072">And I'd like to very quickly fly through a method called</a> <a data-t="2075">TCAV, published in ICML 2018.</a></p>

<p><a data-t="2080">And the idea behind this method is to take this insight</a> <a data-t="2083">that, OK, if things like word embeddings</a> <a data-t="2085">have directions that sort of correspond</a> <a data-t="2088">to things that are meaningful to humans,</a> <a data-t="2091">like capital cities to capitals, maybe</a> <a data-t="2094">there are other types of embeddings</a> <a data-t="2096">that have that same property.</a> <a data-t="2097">And we could use that to our advantage.</a></p>

<p><a data-t="2100">So I'll illustrate this by imagining a question you</a> <a data-t="2105">might ask of an image classifier.</a> <a data-t="2106">Let's say it looks at a picture of a zebra.</a> <a data-t="2109">And you wonder, why did it think that thing was a zebra?</a> <a data-t="2113">So there's sort of a bad answer to this question</a> <a data-t="2115">and a good answer.</a> <a data-t="2116">The bad answer is if it told you something like, oh,</a> <a data-t="2119">it was this particular pixel, or these sets of pixels.</a> <a data-t="2123">Those are not terms that a human can understand.</a> <a data-t="2126">A much better answer would be if it could say something like,</a> <a data-t="2129">oh, it was striped, and that helped.</a></p>

<p><a data-t="2133">Of course, it's very hard to kind of figure out</a> <a data-t="2135">how to extract that answer.</a> <a data-t="2137">However, there is a way to do it.</a> <a data-t="2139">So let's follow what happens if you</a> <a data-t="2142">have a neural network that is recognizing</a> <a data-t="2143">an image, like zebra.</a> <a data-t="2145">So you put in an image.</a> <a data-t="2147">Let's say it's an image of a zebra.</a> <a data-t="2149">And then you put it into your neural network.</a> <a data-t="2152">And what is shown here is a little cartoon</a> <a data-t="2154">of one famous network called Inception.</a></p>

<p><a data-t="2158">And the idea is that in this kind of network,</a> <a data-t="2162">the image gets sort of transformed layer by layer.</a> <a data-t="2166">And you can think of each layer as performing a transformation</a> <a data-t="2169">of Euclidean space.</a> <a data-t="2172">And so if you put something like zebra through it,</a> <a data-t="2175">show it an image, in each of these layers,</a> <a data-t="2178">you can think of the state of the layers</a> <a data-t="2180">as a set of activations of the neurons.</a> <a data-t="2182">And those activations form a vector</a> <a data-t="2184">in high-dimensional space.</a> <a data-t="2185">So you can see how this starts to relate to what we were just</a> <a data-t="2188">looking at.</a></p>

<p><a data-t="2189">And then what's interesting is if you put a whole bunch</a> <a data-t="2192">of images through, you get a bunch of points</a> <a data-t="2194">in this high-dimensional space.</a> <a data-t="2196">And you can start playing this game</a> <a data-t="2198">with meaningful directions.</a> <a data-t="2202">So let's show how this works.</a> <a data-t="2204">So the idea is that let's say there's</a> <a data-t="2206">a concept you're interested in, for example, stripes</a> <a data-t="2209">or the presence of stripes.</a> <a data-t="2212">You don't even have to try to define to the computer</a> <a data-t="2215">in language what stripes is.</a> <a data-t="2216">You could just show it a bunch of examples of striped images.</a></p>

<p><a data-t="2221">And then you put those striped images</a> <a data-t="2223">through your neural network.</a> <a data-t="2225">And then as you see.</a> <a data-t="2226">And one important thing, these are not images of zebras.</a> <a data-t="2229">This is just any stripes, anywhere.</a> <a data-t="2231">We've done this literally with striped ties, and striped</a> <a data-t="2234">shirts, and abstract stripes.</a> <a data-t="2236">That's a good point.</a> <a data-t="2237">And then what you'll get is a kind</a> <a data-t="2239">of cluster of images in this space of activations</a> <a data-t="2242">in an intermediate layer that represents stripes.</a> <a data-t="2245">And then you can also put a bunch of random other images</a> <a data-t="2248">through and train a classifier to distinguish</a> <a data-t="2251">between the stripes and the non-stripes.</a> <a data-t="2253">And that gives you what's called a concept activation</a> <a data-t="2256">vector, a direction in space that represents that concept.</a></p>

<p><a data-t="2260">And then this is what's cool, is that you</a> <a data-t="2262">can start calculating with that.</a> <a data-t="2264">That vector is something you can, for example,</a> <a data-t="2266">take directional derivatives to understand</a> <a data-t="2268">the sensitivity to this concept.</a> <a data-t="2272">So computers like it.</a> <a data-t="2272">But humans like it too, because it's</a> <a data-t="2275">a human-friendly concept of stripes</a> <a data-t="2277">that they're getting an answer in terms of.</a> <a data-t="2280">So this turns out to actually be a very useful technique</a> <a data-t="2283">in general.</a> <a data-t="2285">There's a lot more we can say.</a> <a data-t="2287">But what's really interesting is that you</a> <a data-t="2287">can go from just interpreting or understanding directly</a> <a data-t="2291">to control.</a></p>
