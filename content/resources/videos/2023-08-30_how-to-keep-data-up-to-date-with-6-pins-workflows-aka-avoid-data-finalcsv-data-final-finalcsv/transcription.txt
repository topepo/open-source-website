<h3>Introduction to pins</h3>

<p><a data-t="138">Now, for today's workflow, we're going to focus on Pins.</a> <a data-t="141">And in a second, I'll explain what the Pins package is and how you use it,</a> <a data-t="146">but I thought it might be helpful to first describe a few scenarios</a> <a data-t="150">where you might want to consider using the Pins package for your workflows.</a></p>

<p><a data-t="155">In this first scenario, have you ever created an asset,</a> <a data-t="158">maybe like a piece of data,</a> <a data-t="160">that you thought was the final version of the data,</a> <a data-t="164">but then you had to go back and make some edits</a> <a data-t="166">and create another final version.</a> <a data-t="168">And rather than overwriting the original data set,</a> <a data-t="171">you decide to give it a new name.</a> <a data-t="173">And maybe you go through this process again and again and again,</a> <a data-t="177">and you end up with multiple final versions of a data set,</a> <a data-t="181">and no one knows what the true final version is.</a></p>

<p><a data-t="186">Maybe this data we just talked about is used by many different content types,</a> <a data-t="191">including a Shiny app in R or a </a><a href="/software/quarto/">Quarto</a><a data-t="191"> document in Python,</a> <a data-t="195">and you're worried that the data may not be consistent across all assets.</a> <a data-t="201">Additionally, maybe you want to share this data with multiple people,</a> <a data-t="205">and it's typically much easier and safer to share a Pin</a> <a data-t="209">rather than trying to email files around.</a></p>

<p><a data-t="212">And finally, maybe this data contains sensitive information,</a> <a data-t="216">and you only want to share it with specific people or groups of people.</a> <a data-t="221">And this is actually possible when using pins in combination with Posit Connect.</a></p>

<blockquote><a data-t="230">The best way to think of pins, in my opinion, is to think of a corkboard.</a> <a data-t="234">Similar to how you can pin a note to a corkboard,</a> <a data-t="237">you can pin an R or Python object to a board</a> <a data-t="241">where you and others can access it.</a></blockquote>
