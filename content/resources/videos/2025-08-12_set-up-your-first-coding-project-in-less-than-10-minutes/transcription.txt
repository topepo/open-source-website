<p><a data-t="0">This is how to set up your very first coding project using </a><a href="/software/positron/">Positron</a><a data-t="0"> in less than 10 minutes.</a> <a data-t="4">The first step is to create a folder and open it.</a> <a data-t="6">You can do that using the terminal commands or just the normal file picker structure.</a> <a data-t="10">You'll be prompted to pick your python interpreter.</a></p>

<p><a data-t="12">Step 2 is to install your commonly used packages like </a><a href="/software/shiny-r/">shiny</a><a data-t="12"> and pandas using the pip command.</a> <a data-t="17">Once those have installed, you can then use shiny create to create your scaffold of the</a> <a data-t="20">shiny app.</a> <a data-t="21">This will set up your app.py with starter UI and server code.</a></p>

<p><a data-t="24">Step number 3 is to run it.</a> <a data-t="26">In Positron, you can click the play button on the file itself, and it will automatically</a> <a data-t="30">run your shiny app in the viewer pane, or you can run this command from the terminal.</a> <a data-t="34">Using the flag for reload ensures that any edits you make in the app.py folder will show</a> <a data-t="37">up immediately.</a></p>

<p><a data-t="38">And then for step 4, edit your file and watch it update in real time.</a> <a data-t="42">Anytime you make a change and save the file, it will automatically reload in the app.</a></p>

<p><a data-t="46">And then for step 5, you can drop in a breakpoint for any debugging you need.</a> <a data-t="49">Click on the left of the line number to set up a breakpoint, then click the dropdown arrow</a> <a data-t="52">next to run, select debug shiny app.</a> <a data-t="55">The app will pause right where you placed it, and now you can step through the lines,</a> <a data-t="58">inspect variables, and see the call stack.</a></p>
