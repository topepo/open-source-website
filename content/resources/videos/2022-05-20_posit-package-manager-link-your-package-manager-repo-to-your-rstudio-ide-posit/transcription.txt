<p><a data-t="0">Hey Rockstar, do you know if your R and Python packages are out for delivery?</a> <a data-t="6">Let me show you how to know.</a> <a data-t="8">Head over to </a><a href="/software/rstudio/">RStudio</a><a data-t="8"> Package Manager.</a> <a data-t="11">You'll probably land on the top repo your system administrator has set up for you.</a></p>

<p><a data-t="14">In the top navigation bar, head over to Setup.</a> <a data-t="18">Scroll down below the calendar, click on the Change link.</a> <a data-t="23">Select the operating system that your IDE is running on.</a> <a data-t="26">In my case that's Ubuntu 20.04.</a> <a data-t="29">Select OK.</a></p>

<p><a data-t="31">Select the Binary button.</a> <a data-t="32">This ensures you'll be installing precompiled binary versions of the packages, saving you</a> <a data-t="37">hours of time.</a> <a data-t="39">Click the Copy button to copy this unique URL to your clipboard.</a></p>

<h3>Connecting to the RStudio IDE</h3>

<p><a data-t="44">Then head back over to your RStudio IDE.</a> <a data-t="48">In the Tools menu at the top, go down to Global Options.</a> <a data-t="53">In the left sidebar, click on Packages.</a> <a data-t="56">And then the Change button.</a> <a data-t="59">And then paste in the URL that you copied from Package Manager.</a> <a data-t="63">Click OK and OK.</a></p>

<p><a data-t="65">That's it.</a> <a data-t="66">That's the simplest way to connect RStudio Package Manager to your RStudio IDE.</a> <a data-t="71">It can be a lot more complicated than that.</a> <a data-t="73">There's multiple ways to do this, but this is your first step.</a> <a data-t="76">Give it a shot.</a> <a data-t="77">Bring those packages home.</a></p>
