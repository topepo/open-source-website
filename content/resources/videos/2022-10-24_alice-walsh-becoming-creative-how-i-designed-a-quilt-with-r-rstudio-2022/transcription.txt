<h3>Quilting as an art form</h3>

<p><a data-t="111">So now quilting.</a> <a data-t="111">For those of you who don't know what a quilt is, it's a blanket, and quilting has been</a> <a data-t="114">in practice for thousands of years.</a> <a data-t="117">And what makes a quilt is that you have three layers.</a> <a data-t="119">You have a top, you have an insulating middle layer, and you have a back layer.</a> <a data-t="124">And to quilt, you connect these layers together, usually with thread, and this can be very</a> <a data-t="129">decorative.</a> <a data-t="131">It's an art form.</a></p>

<p><a data-t="131">And personally, I got really excited about making quilts as art, and particularly the</a> <a data-t="137">modern quilting movement.</a> <a data-t="144">Quilting is just fabric, you cut up, you sew together, and then you sew it together</a> <a data-t="148">again.</a></p>

<p><a data-t="155">So back to modern quilting, this is an example of a beautiful modern quilt.</a> <a data-t="160">It has these organic shapes made out of fabric.</a> <a data-t="164">This one I also love, extremely different, but it has this beautiful composition that</a> <a data-t="167">draws your eye around the frame.</a> <a data-t="167">Quilts can also play with color, like this example, which is really challenging to do,</a> <a data-t="171">given that you don't have every color in the rainbow available to you in fabric.</a> <a data-t="175">Quilts can also have political messages.</a> <a data-t="178">Or quilts can be personal, like this is a self-portrait, someone made in fabric.</a></p>

<h3>The creative process: generating ideas</h3>

<p><a data-t="205">So the first step in my creative process is to come up with too many ideas.</a> <a data-t="210">I'm somebody that I like to come up with one idea, and then execute the crap out of it.</a> <a data-t="215">Like don't keep thinking about other things, but my new method is that I try to come up</a> <a data-t="219">with more ideas than I think I need.</a> <a data-t="221">I have an idea quota, where I come up with 10 ideas, 40 ideas, 50 ideas.</a> <a data-t="225">From that, you end up with new ideas that you had never originally thought on day one,</a> <a data-t="230">and also takes you in directions that lets you better curate your ideas.</a></p>

<p><a data-t="234">And what I ended up doing with this project was coming up with a solution where I could</a> <a data-t="239">fuse two domains.</a> <a data-t="241">And I had just taken a drawing class, and I learned about the concept of two-point perspective.</a> <a data-t="246">Two-point perspective is a way to make a 3D visualization in two dimensions, like these</a> <a data-t="252">cuboid shapes.</a> <a data-t="253">And I also know about R, and I thought, well, I'm really excited about the things that are</a> <a data-t="257">happening with generative art using R.</a> <a data-t="260">Could I create some sort of landscape in this way, a 3D, 2D production of a 3D landscape</a> <a data-t="265">using R, using something like geom polygon?</a> <a data-t="268">And so with that, I was really happy with that idea, and I started to go forward.</a></p>

<h3>Improvisation and research</h3>

<p><a data-t="274">So step two was I did a lot of improvisation.</a> <a data-t="277">So I can start to learn, how could I make this visualization in R?</a> <a data-t="280">How could I mess around with the dimensions?</a> <a data-t="283">How many shapes I wanted, the size of these, the overall aspect ratio of the colors?</a></p>

<p><a data-t="290">And then I had to do a lot of research, because the fact is that the design that I had chosen</a> <a data-t="297">is not easily convertible to fabric, which is not something that I had planned for.</a> <a data-t="302">So I had to come up with some techniques that would allow me to construct my design in fabric,</a> <a data-t="307">and that is where I want to talk to you a little bit about a technique I really like,</a> <a data-t="311">which is called foundation paper piecing.</a> <a data-t="313">It's a little bit of a rabbit hole, I'll go down with you.</a></p>

<p><a data-t="316">So this is a technique where you create a design, and you print it on a piece of paper.</a> <a data-t="321">So you have a design like this, and you actually sequentially sew fabric to that design.</a> <a data-t="326">So this is what the back would look like, and then the front looks like this.</a> <a data-t="330">And this is an amazing technique, because it allows you to do very, very</a> <a data-t="334">precise piecing of fabrics together.</a> <a data-t="337">Those lines are all perfectly parallel.</a> <a data-t="339">You can have very, very small pieces.</a> <a data-t="341">And this lends itself very much to the technique and the design that I wanted to make.</a></p>

<p><a data-t="346">But there's a problem in that this technique also imparts a new geometrical constraint</a> <a data-t="351">in the type of a design you can create.</a> <a data-t="354">So you can only add one piece of fabric at a time, and you can only add one seam at a time,</a> <a data-t="359">which seems simple, but it does impose a constraint on the amount of design that you can do.</a> <a data-t="363">So for something like my design, I'm in trouble, because this is not pieceable.</a> <a data-t="368">This design with this Y junction in the middle, I cannot do, because I would have to add two</a> <a data-t="372">pieces of fabric at the same time.</a> <a data-t="374">It doesn't work.</a></p>

<p><a data-t="376">So the way that people who design these patterns work is it's basically like a Sudoku puzzle.</a> <a data-t="381">It's all done by hand, where you look at this, and I say, well, I can't do this in one block,</a> <a data-t="386">so I'll split it into two, and then I have to design where my seams will go,</a> <a data-t="391">and then I also have to determine the correct order,</a> <a data-t="393">because if I don't add these pieces in the correct order, it won't work.</a></p>

<p><a data-t="398">And I really thought a lot about, okay, well, is there a way I could use R</a> <a data-t="402">or some other, you know, framework to help me do this?</a> <a data-t="405">And I found a wonderful paper that was published just last year by Mackenzie Leak</a> <a data-t="408">and others at Stanford University, which actually gets into the mathematical foundation for doing</a> <a data-t="414">this, and I really recommend, if you're interested, to go ahead and read this paper.</a> <a data-t="419">They've also got some YouTube videos and even a web app that you can use to help design your</a> <a data-t="423">own designs.</a></p>

<p><a data-t="424">And so I don't really want to talk about graph theory, although I do a little bit,</a> <a data-t="429">which is that these designs are then represented as hypergraphs, where each piece is a node in</a> <a data-t="434">the graph, and each seam is an edge, and this allows you, then, to determine if your design</a> <a data-t="439">is peaceable and also automatically derive the order that you would add the pieces.</a> <a data-t="444">It's really cool.</a> <a data-t="445">So if we take a complicated design, like the one I made of the R logo on the left,</a> <a data-t="450">that's actually four pieces, and I can represent them as four graphs, and from that, I can</a> <a data-t="454">determine that my design was good because it was peaceable, and also the order that I need</a> <a data-t="459">to add the pieces.</a> <a data-t="460">So that was great.</a></p>

<h3>Execution and sharing the work</h3>

<p><a data-t="463">Except for now I have to make the quilt.</a> <a data-t="466">And execution is a really hard part of creativity, because if you just have a lot of ideas and</a> <a data-t="470">you don't actually follow through and finish them, it's really hard to make the quilt.</a></p>

<h3>Metaphors for creative work</h3>

<p><a data-t="711">Okay, so I want to now talk about a couple tortured metaphors</a> <a data-t="718">that I've learned from other people that I think apply to my work in computational research.</a> <a data-t="723">And hopefully, also will resonate with some of you</a> <a data-t="726">in the kind of work that you do and how you could try to be more creative.</a></p>

<p><a data-t="730">So this first example, I learned from a colleague of mine at BMS, Joe Sostakowsky.</a> <a data-t="734">And the question is, would you rather be a pizza deliverer or would you rather be a </a><a href="/software/plumber/">plumber</a><a data-t="734">?</a> <a data-t="740">Okay, how do you get pizza?</a> <a data-t="744">I order pizza a lot, and it's great.</a> <a data-t="746">I can get on my phone, I can say what toppings I want, how many pizzas.</a> <a data-t="751">And then I get like a photo of the pizza on my doorstep.</a> <a data-t="755">It's incredible, right?</a> <a data-t="756">It works really, really well for food delivery.</a></p>

<p><a data-t="758">But I've also been on teams where I'm a data analyst.</a> <a data-t="762">And I'm asked for things in the same way.</a> <a data-t="765">Somebody goes on and they order a data pizza.</a> <a data-t="767">They say, I need this dashboard with these features with this data.</a> <a data-t="771">And I don't think that's a very effective way to do data work.</a> <a data-t="774">I don't think that's a good way for you to get insights from data.</a> <a data-t="778">And it doesn't leave any room for creativity</a> <a data-t="780">or trying to get to what are really the right questions you should be asking.</a></p>

<p><a data-t="784">So a slight iteration on that is to think about a plumber.</a> <a data-t="787">So a plumber is a subject matter expert, right?</a> <a data-t="790">I have a plumber come to my house, and they talk to me about what my problems are.</a> <a data-t="794">They bring their own toolkit.</a> <a data-t="795">They have their own expertise.</a> <a data-t="797">And they try to understand really what my needs are.</a> <a data-t="799">And they work on a solution that will meet my needs.</a></p>

<p><a data-t="803">OK, so if you don't like that one, that's fine.</a> <a data-t="806">I have another one.</a> <a data-t="808">So this one is, would you rather be a detective or a criminal?</a> <a data-t="813">And it's a trick question.</a> <a data-t="814">You actually want to be both.</a> <a data-t="816">So this is actually one from Albert Einstein.</a> <a data-t="820">And so this is about explaining why science is so difficult.</a> <a data-t="823">So if you take a detective, like a Sherlock Holmes,</a> <a data-t="826">there's a murder's been committed.</a> <a data-t="828">And you collect clues, and you have hypotheses.</a> <a data-t="830">Like, it's a jealous girlfriend.</a> <a data-t="832">It's a ex-colleague or whatever.</a> <a data-t="834">And you put together all these clues.</a> <a data-t="836">And then you discover, like, oh, actually, it was a poisonous jellyfish that killed the person.</a> <a data-t="840">Like, that's how a detective works, right?</a></p>

<p><a data-t="843">But the key is that a detective knows what crime they're solving.</a> <a data-t="846">The problem is really well-defined.</a> <a data-t="848">This person's been murdered.</a> <a data-t="849">This has been stolen.</a> <a data-t="851">I know how to solve it.</a> <a data-t="852">But the difference is, for a scientist, you have to also come up with the questions.</a> <a data-t="857">So in that way, you have to commit the crime.</a> <a data-t="860">And you have to perform the investigation.</a> <a data-t="863">And I find this is a really apt metaphor for the work that I do.</a> <a data-t="866">Because you have to constantly be asking, am I asking the right questions?</a> <a data-t="870">Is this the important line of research that I should be following?</a> <a data-t="873">And I think that's a really, really important part of doing science</a> <a data-t="877">that requires you to be creative.</a></p>

<blockquote><a data-t="866">Because you have to constantly be asking, am I asking the right questions?</a> <a data-t="870">Is this the important line of research that I should be following?</a> <a data-t="873">And I think that's a really, really important part of doing science</a> <a data-t="877">that requires you to be creative.</a></blockquote>
