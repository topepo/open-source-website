<p><a data-t="0">Let's show you how to deploy your data science content to Posit Connect Cloud.</a> <a data-t="5">In this example, we'll be deploying a </a><a href="/software/shiny-python/">Shiny for Python</a><a data-t="5"> application,</a> <a data-t="8">but Posit Connect Cloud is not limited to just this framework.</a></p>

<h3>Setting up the GitHub repository</h3>

<p><a data-t="14">Deploying on Connect Cloud starts with the GitHub repository.</a> <a data-t="19">Here, we'll create an empty repository named dashboard,</a> <a data-t="24">and then clone it down to our computer.</a></p>

<p><a data-t="30">Next, we'll create our </a><a href="/software/shiny-r/">Shiny</a><a data-t="30"> for Python application.</a> <a data-t="34">Here, we'll use an existing Shiny for Python template</a> <a data-t="37">and save it into our cloned repository.</a></p>

<p><a data-t="48">We'll follow the instructions to install the template dependencies</a> <a data-t="50">and run it in </a><a href="/software/positron/">Positron</a><a data-t="50">.</a></p>

<h3>Running the app locally</h3>

<p><a data-t="60">Let's run this application locally first.</a> <a data-t="75">We'll open the app.py file and click the play button from the Shiny VS Code extension.</a></p>

<p><a data-t="90">We can see the reactivity of the application when we change the checkboxes.</a></p>

<h3>Deploying to Connect Cloud</h3>

<p><a data-t="97">Next, we need to get all the code to our GitHub repository.</a> <a data-t="110">Now that the code is on GitHub, let's go to Connect Cloud and deploy our application.</a></p>

<p><a data-t="116">After you log in and connect your GitHub account to Connect Cloud,</a> <a data-t="119">we can click publish, Shiny, and select our repository.</a></p>

<p><a data-t="130">Our current application is on the main branch and is named app.py.</a> <a data-t="146">We then hit publish.</a></p>

<p><a data-t="165">Our app is now published at the URL.</a></p>

<h3>Updating and sharing your content</h3>

<p><a data-t="176">Now, let's make a change to the capitalization of the dashboard title.</a> <a data-t="195">The changes will automatically republish the application.</a> <a data-t="202">You do not need to click the republish button by default.</a></p>

<p><a data-t="210">In the content settings, by clicking on it here,</a> <a data-t="214">you have the option to turn off auto-republish</a> <a data-t="216">and also the option to give your content a custom URL.</a> <a data-t="220">Posit Connect Cloud is an easy way to deploy straight from an existing GitHub repository</a> <a data-t="226">and share your data science work.</a></p>
