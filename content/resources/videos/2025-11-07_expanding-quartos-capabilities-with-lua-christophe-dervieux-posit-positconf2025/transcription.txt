<h3>Quarto extensions for reuse and distribution</h3>

<p><a data-t="908">And it's a bit natural to ask yourself,</a> <a data-t="911">like you have this one-shot filter that you've made.</a> <a data-t="913">And maybe you want to reuse it across your project.</a> <a data-t="916">Or maybe you are working in an organization</a> <a data-t="918">where different users have the same problem.</a> <a data-t="921">So the question of reusing these Lua filters makes sense.</a> <a data-t="924">And this is, at that point, that you</a> <a data-t="925">can consider Quarto extensions.</a> <a data-t="928">They are a way to reuse and distribute</a> <a data-t="932">some feature of Quarto.</a> <a data-t="934">So filter is one of them.</a> <a data-t="936">And you see that Quarto extension</a> <a data-t="939">can be of several forms.</a> <a data-t="940">There are also shortcodes.</a> <a data-t="941">So with the knowledge of Lua function,</a> <a data-t="943">you can also write shortcodes.</a> <a data-t="945">And there is other types.</a> <a data-t="947">You can do extension for custom formats or brand.</a> <a data-t="949">And they don't require knowing Lua at all.</a> <a data-t="951">But they are a way to ship that to several projects</a> <a data-t="954">or several users.</a></p>
