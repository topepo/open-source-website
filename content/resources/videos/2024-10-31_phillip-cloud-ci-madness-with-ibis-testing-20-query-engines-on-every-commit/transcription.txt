<h3>Managing complexity</h3>

<p><a data-t="290">So how do we, how do we manage this complexity?</a> <a data-t="295">Um, well, there's, there's like basically one high-level goal, which is to iterate quickly.</a> <a data-t="301">And that's sort of project dependent, but we found that there are a few things that really matter when trying to do this.</a> <a data-t="307">And one is that your environment setup needs to be fast, because if you're going to run this stuff in continuous integration, like say on GitHub actions, you don't want to spend a bunch of your time solving dependency constraints.</a></p>

<h3>Measuring CI performance</h3>
