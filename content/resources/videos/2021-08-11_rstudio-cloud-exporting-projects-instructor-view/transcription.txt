<p><a data-t="0">There are times that you may want to export your work from </a><a href="/software/rstudio/">RStudio</a><a data-t="0"> Cloud, and we've done</a> <a data-t="7">our best to make it as easy as possible for you.</a></p>

<p><a data-t="11">Navigating to our workspace from within RStudio Cloud, we see that there are a few projects</a> <a data-t="15">listed.</a> <a data-t="17">To the right of each project is an export button, and by clicking on this, RStudio Cloud</a> <a data-t="22">initiates the creation of a project file that we can download.</a></p>

<p><a data-t="26">Once complete, the pop-up shows us a download button.</a> <a data-t="30">Clicking the download button downloads a zip file with all of the associated project files</a> <a data-t="34">that we can now run on our local machine.</a></p>
