<h3>The Baltimore Community Data Science course</h3>

<p><a data-t="177">So we were left with maybe wanting to try an experiment.</a> <a data-t="177">How could we train students</a> <a data-t="183">in more ethical conscious practice and also empower community organizations?</a> <a data-t="188">How could we get community organizations with data science students to work together</a> <a data-t="193">to tell data-empowered stories?</a></p>

<p><a data-t="197">So we embarked on this experiment, myself and Ava Hoffman, who's in the audience, and another</a> <a data-t="203">colleague, Michael Rosenblum, to create a course called the Baltimore Community Data Science Course,</a> <a data-t="209">or lovingly BCDS, at the Johns Hopkins Bloomberg School of Public Health.</a> <a data-t="216">And we did this in partnership with SOURCE, which is the Community Engagement and Service</a> <a data-t="220">Learning Center at Hopkins, which is focused on doing engagement that actually strives for</a> <a data-t="226">social change, authentic relationships, and redistributing power.</a></p>

<p><a data-t="231">So Michael and I got specialized training that we continue to get from SOURCE to really learn how to</a> <a data-t="239">engage with the community in a mindful way.</a> <a data-t="241">So the idea was we wanted to do good while teaching the students to do good.</a> <a data-t="248">So work with a community-based organization to help them with a data science project,</a> <a data-t="252">but also have the students learn to perform data science in a more intentional and reflective way.</a></p>

<p><a data-t="260">So our philosophy was that the community would come first, and data science would come second.</a> <a data-t="266">And ultimately, we would empower those organizations to tell their own data science story.</a></p>

<h3>Course structure and preparation</h3>

<p><a data-t="389">So this is an actual slide from one of our slide decks, where on the first day,</a> <a data-t="393">we tell them that this will feel very different.</a> <a data-t="393">And the class structure looks like this.</a> <a data-t="393">It's a</a> <a data-t="400">16-week course, and we spend a lot of time preparing the students to work with the community</a> <a data-t="405">before we let them do so.</a> <a data-t="405">Then we have them work on the project and have them think about</a> <a data-t="410">sustainability.</a></p>

<p><a data-t="417">So the first week, we talk about the historical and current context of Baltimore</a> <a data-t="418">and Johns Hopkins and the community.</a> <a data-t="422">It's important that the students learn about the</a> <a data-t="422">context of who they're working with.</a> <a data-t="428">Then we have them learn about critical service learning,</a> <a data-t="428">the idea that we're not doing typical community service.</a> <a data-t="433">This is a different type where we're</a> <a data-t="433">really reflective on our potential impact long-term.</a> <a data-t="440">And this involves critical reflection.</a> <a data-t="440">So they need to learn this process of considering our perspective and assumptions and where it comes</a> <a data-t="446">from.</a> <a data-t="446">And sustainable design.</a> <a data-t="446">We want to create change that lasts.</a> <a data-t="451">We don't want to</a> <a data-t="451">create a product for a CBO, leave them, and they can't use it.</a></p>

<p><a data-t="457">And also think about things like data privacy, security,</a> <a data-t="460">transparency, so practices that would support the students to do this in a safe manner.</a></p>

<p><a data-t="467">And then, and only then, are the students allowed to work on the projects.</a> <a data-t="472">So this is really about making a commitment because we have to produce something that's</a> <a data-t="477">useful for the community organizations.</a> <a data-t="477">And so we spend a lot of time talking to the organizations</a> <a data-t="482">ahead of time to make sure that this is actually feasible.</a> <a data-t="488">And we've done this for two years.</a> <a data-t="488">We've</a> <a data-t="488">worked with a variety of organizations.</a> <a data-t="493">The first year, we worked with a program called HeartSmiles,</a> <a data-t="493">which does leadership opportunities for underserved youth.</a> <a data-t="498">The No Boundaries Coalition,</a> <a data-t="498">which was working on a narrative on gun violence.</a> <a data-t="503">And the Baltimore Transit Equity Coalition,</a> <a data-t="503">which works for equitable access to public transit.</a> <a data-t="509">In the second year, we again worked</a> <a data-t="509">with HeartSmiles.</a> <a data-t="514">We also worked with the Tool Bank in Baltimore, which provides access for</a> <a data-t="514">tools for community projects.</a> <a data-t="519">And the League of Women Voters, which works to enhance voter</a> <a data-t="519">participation.</a></p>

<h3>Five major lessons</h3>

<p><a data-t="599">So through this process, we learned five major lessons.</a> <a data-t="600">The first one is that context really matters, especially in a place like Baltimore.</a> <a data-t="605">There's</a> <a data-t="605">a long history of structural racism, and that impacts the way things work today.</a> <a data-t="611">And our students</a> <a data-t="611">need to know that.</a> <a data-t="615">Many of our students, it's the first time that they're learning about it.</a> <a data-t="616">They may be from outside the United States, or they may be from some part of the U.S. that has</a> <a data-t="620">a different experience.</a> <a data-t="620">And this context leads them to more appropriately work with our CBOs.</a> <a data-t="627">So never assume that your students don't necessarily need to learn information.</a></p>

<p><a data-t="631">That's the takeaway here.</a> <a data-t="631">Number two is that people come first.</a> <a data-t="631">So data science products</a> <a data-t="637">need to meet people where they are, and they need to meet their actual need.</a> <a data-t="643">And it's about what's useful to the CBO.</a> <a data-t="643">It's not about what's cool for the students to work on,</a> <a data-t="648">which was also a source of tension.</a> <a data-t="648">So students often wanted to create something more exciting</a> <a data-t="655">that would involve more infrastructure that the CBOs would need to learn.</a> <a data-t="660">So for example,</a> <a data-t="660">developing an LLM is not necessarily the first thing that's a priority for the CBOs.</a></p>

<p><a data-t="667">So this requires flexibility, and we have to be flexible and ready for changes.</a></p>

<p><a data-t="677">Next is critical reflection.</a> <a data-t="677">So we taught the students to reconsider, question, challenge why</a> <a data-t="683">things are the way that they are.</a> <a data-t="683">Why do we need these CBOs to even be working on the problems</a> <a data-t="687">that they're working on?</a> <a data-t="694">And consider how our perspective came to be what it is, how CBO's</a> <a data-t="694">perspective is what it is.</a> <a data-t="699">And ultimately recognizing and respecting that we're</a> <a data-t="699">never fully going to understand one another, but striving to is beneficial.</a></p>

<p><a data-t="705">So because we're working with STEM students, and we ourselves are also from STEM, we like things</a> <a data-t="711">to be kind of concrete.</a> <a data-t="711">And we were working with some people that had trained us about critical</a> <a data-t="716">reflection in a much more mushy kind of way.</a> <a data-t="716">And so we decided to make a framework that would be</a> <a data-t="722">more comfortable for the STEM students.</a> <a data-t="722">And it's here if you're interested in using it.</a> <a data-t="727">The idea</a> <a data-t="727">is that we would first have the students think about what was resonating with them.</a> <a data-t="732">What do they</a> <a data-t="732">feel curious about?</a> <a data-t="739">Second step is to think about the context.</a> <a data-t="739">That could be from time, space.</a> <a data-t="740">Consider different viewpoints.</a> <a data-t="740">How might different people think about this?</a> <a data-t="744">Assess how changes may have happened over time.</a> <a data-t="744">Challenge assumptions.</a> <a data-t="750">And then finally evaluate both yourself and how others are experiencing this.</a></p>

<p><a data-t="758">And as an example of one thing that we did is we have them reflect on their reflections after we</a> <a data-t="764">learn this process with them.</a> <a data-t="773">And so really learning critical reflection is actually a process.</a> <a data-t="773">And they find that activity of reflecting on how they initially did such a thing very meaningful.</a></p>

<p><a data-t="782">Number four is that challenges will happen.</a> <a data-t="782">Mistakes will happen.</a> <a data-t="787">Anytime you're working on anything particularly sensitive, mistakes will happen.</a> <a data-t="787">And it's about</a> <a data-t="792">mitigating them and preventing them.</a> <a data-t="792">And of course, taking responsibility and teaching those lessons</a> <a data-t="797">to students is really valuable.</a> <a data-t="797">So one of our examples is we are working with these underserved</a> <a data-t="804">youth and teaching them about data science.</a> <a data-t="804">And we had an experience where some jargon came up</a> <a data-t="809">that could be unexpectedly offensive, which we did not realize.</a> <a data-t="809">But luckily, we were working with</a> <a data-t="814">the CBO, and they were checking all of our materials ahead of time.</a> <a data-t="818">And we learned that</a> <a data-t="818">the abbreviation for POE, which is one of the platforms for AI, could be an offensive term.</a></p>

<p><a data-t="832">Next is sustainability and impact matters.</a> <a data-t="832">So much like a trail,</a> <a data-t="832">we want to leave things better than how we found it.</a> <a data-t="838">So we want to enable people to best continue</a> <a data-t="838">without us as possible.</a> <a data-t="845">So as an example, we helped make a mock-up website and said, you know,</a> <a data-t="845">this is something you could do.</a> <a data-t="850">Now we have all this data that you could work with.</a> <a data-t="850">And then</a> <a data-t="850">the organization took it and did a really incredible job.</a> <a data-t="856">And so we want to leave space</a> <a data-t="856">for the CBO to use that data themselves.</a></p>
