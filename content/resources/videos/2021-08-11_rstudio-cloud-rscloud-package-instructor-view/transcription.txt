<p><a data-t="0">As R users, we know that sometimes code is the best way to handle things, and in this</a> <a data-t="8">video walkthrough, we'll be focused on setting up our access to the rscloud package.</a></p>

<h3>Setting up client credentials</h3>

<p><a data-t="14">To get started using the rscloud package, we first need to set up our client credentials,</a> <a data-t="20">and we do this from within </a><a href="/software/rstudio/">RStudio</a><a data-t="20"> Cloud.</a> <a data-t="23">After logging in to our RStudio Cloud account, we can click on our icon or photo in the top</a> <a data-t="28">right to access the user panel, and then select Client Credentials.</a> <a data-t="34">This will take us to a page with any existing credentials we might have.</a> <a data-t="38">To create new credentials, click on Plus New Credentials in the top right.</a></p>

<p><a data-t="44">This will open a popup where you can name your credentials.</a> <a data-t="48">After naming your credentials, click OK to see and copy your generated credentials.</a> <a data-t="53">You will only be shown the credentials once, so do be sure to copy them and store them</a> <a data-t="57">in a secure place before closing the popup window.</a> <a data-t="60">Be careful not to share or commit your credentials to Git.</a></p>

<h3>Configuring the R environment</h3>

<p><a data-t="65">The recommended method for setting your credentials is by editing your R Environ file, and we</a> <a data-t="70">can do this using the </a><a href="/software/usethis/">usethis</a><a data-t="70"> package.</a> <a data-t="74">From within RStudio Cloud, we can go to a project, and by running usethis, edit renviron</a> <a data-t="81">in the console.</a> <a data-t="83">This should then open your R Environ file, where you can paste and save your client credentials</a> <a data-t="88">before restarting R.</a></p>

<p><a data-t="90">After setting up your credentials in your R Environ file, you can install the development</a> <a data-t="95">version of the rscloud package by running the following commands in the console.</a> <a data-t="100">install .packages remote and remotes install github rstudio rscloud</a> <a data-t="107">Once this is done, you have the rscloud package available and ready to use in RStudio Cloud.</a></p>

<h3>Using the rscloud package</h3>

<p><a data-t="114">From here, we can do things like run rscloud underscore space underscore list, which will</a> <a data-t="119">return a table with all of the spaces available within our RStudio Cloud account, and we can</a> <a data-t="125">use a space underscore id from this table to look at the number of users and projects.</a> <a data-t="132">For example, when I run rscloud underscore space using the space id 81721, I can see</a> <a data-t="139">that there are 327 users and 946 projects within this shared workspace.</a></p>
