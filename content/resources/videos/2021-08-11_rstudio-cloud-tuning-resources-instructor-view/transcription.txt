<p><a data-t="0">Depending on your </a><a href="/software/rstudio/">RStudio</a><a data-t="0"> Cloud pricing plan, you may have access to the ability to tune</a> <a data-t="7">your project resources.</a> <a data-t="9">To access project resources, we'll need to be within a project or an assignment on RStudio</a> <a data-t="14">Cloud.</a> <a data-t="15">From here, we can click on the Settings icon at the top of our project and navigate to</a> <a data-t="20">the Resources tab.</a></p>

<p><a data-t="22">From here, we can change the amount of memory allocation, CPU allocation, and or the background</a> <a data-t="28">execution limit.</a></p>

<p><a data-t="30">RStudio Cloud's pricing is calculated based on project hours and not project runtime.</a> <a data-t="37">As such, note that changing our memory allocation and or changing our CPU allocation will change</a> <a data-t="43">the project hours consumed by the project based on one hour of project runtime.</a></p>
