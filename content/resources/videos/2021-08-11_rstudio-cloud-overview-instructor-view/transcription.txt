<h3>The challenge of local installation</h3>

<p><a data-t="16">When you're getting ready to teach R with RStudio, you have two main options.</a> <a data-t="20">One option is to walk learners through downloading and installing both R and RStudio on their local machine,</a> <a data-t="27">after which you then have to walk through the installation of all the necessary packages, as well as loading the packages.</a> <a data-t="32">And if you're using Git, you've got to install Git.</a> <a data-t="32">And more often than not, this process results in spending a significant amount of time</a> <a data-t="40">troubleshooting unique installation issues across a variety of operating systems.</a></p>

<h3>RStudio Cloud as an alternative</h3>

<p><a data-t="45">Another option is to use RStudio Cloud.</a> <a data-t="45">RStudio Cloud is a lightweight cloud-based solution that allows you to teach data science using the</a> <a data-t="54">RStudio IDE directly from your browser.</a> <a data-t="57">With RStudio Cloud, Git works out of the box, and you can avoid the majority of the</a> <a data-t="61">installation issues you're likely to encounter when you have learners download to their personal computer.</a> <a data-t="66">What's more, you don't need any system administration experience or IT support to get your course set up.</a> <a data-t="73">Let's take a look.</a></p>

<blockquote><a data-t="45">RStudio Cloud is a lightweight cloud-based solution that allows you to teach data science using the</a> <a data-t="54">RStudio IDE directly from your browser.</a> <a data-t="57">With RStudio Cloud, Git works out of the box, and you can avoid the majority of the</a> <a data-t="61">installation issues you're likely to encounter when you have learners download to their personal computer.</a></blockquote>

<h3>Workspaces and base projects</h3>

<p><a data-t="73">Once you log into RStudio Cloud, you have a variety of tools available to you.</a> <a data-t="79">In the menu on the left, we can take a look at workspaces.</a> <a data-t="79">You'll always have your own personal workspace, and you can create additional workspaces for each of your courses, like this.</a></p>

<p><a data-t="108">To avoid having students each download packages to their computer, we can set up all of the projects within our workspace using a default set of files and packages by creating a base project.</a> <a data-t="117">This means we can install all of the packages used throughout our course, along with any files we want our learners to have access to by default.</a> <a data-t="124">And we can update this base project throughout our course as needed.</a></p>

<h3>Managing members and projects</h3>

<p><a data-t="139">Let's check out what's available to us within a workspace.</a> <a data-t="142">One of the first things we can do is add members to our workspace and set their default permissions, either through invitations or by using a link.</a> <a data-t="150">In addition to adding learners, this is a fantastic way to get any co-teachers and teaching assistants added to your workspace so that they can assist with adding learners and setting up the workspace.</a></p>

<p><a data-t="160">We can also create a new RStudio project with the click of a button, which will take us to an R project that looks like what we'd see in the desktop version of the RStudio IDE.</a> <a data-t="169">From here, we can share the project with our learners and even convert it into an assignment.</a></p>

<h3>Assignments and learner support</h3>

<p><a data-t="174">When we create an assignment as educators, RStudio Cloud automatically makes a copy for our learners and flags the project with the assignment tag from our perspective,</a> <a data-t="183">while showing learners either a start or continue button for each assignment in their workspace.</a></p>

<p><a data-t="189">Another feature in RStudio Cloud is the ability for educators to look at a learner's assignment.</a> <a data-t="194">This means that we can run a learner's work on RStudio Cloud and better help them to identify and debug any errors.</a></p>
