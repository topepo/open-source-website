<h3>Interactive theming tool</h3>

<p><a data-t="179">BSLib also provides an interactive theming tool, which allows you to quickly try out</a> <a data-t="184">different theme settings and seeing how that theme renders in real time.</a> <a data-t="190">This theming tool can be used with any Shiny application by calling the bsthemer function.</a> <a data-t="197">The only requirements is that you call the bsthemer function inside of the server function</a> <a data-t="203">and that you're using this interactive themer with Bootstrap 4.</a> <a data-t="208">Also keep in mind that if you're running this themer locally, any of the interactive changes</a> <a data-t="212">that you make are captured with R code and emitted to your R console so that you can</a> <a data-t="217">copy-paste those changes into your theme creation.</a></p>

<h3>Using theming tools with R Markdown</h3>

<p><a data-t="221">Now if you'd like to use these theming tools with R Markdown, the first thing to know is</a> <a data-t="225">that you can pass these theme settings into the theme parameter of an HTML document.</a> <a data-t="231">In addition, if you'd like to theme your document with the interactive theming tool, you'll</a> <a data-t="235">have to first add a Shiny runtime to your document's YAML matter and then call the bsthemer</a> <a data-t="241">function inside of the server side code.</a> <a data-t="244">Now if you'd like these real-time changes in the CSS on the page to influence the R</a> <a data-t="249">plots rendered by the document, you'll first have to enable thematic and then also make</a> <a data-t="255">sure that you're rendering your plots through Shiny's render plot.</a></p>
