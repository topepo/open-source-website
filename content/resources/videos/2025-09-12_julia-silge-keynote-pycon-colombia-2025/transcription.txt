<h3>The garden of forking paths</h3>

<p><a data-t="383">A result of this kind of iterative, exploratory nature of data work is that we end up in a situation</a> <a data-t="391">where we are making a lot of decisions.</a> <a data-t="395">We're making a lot of decisions, and the one decision we make depends on the last decision we made.</a> <a data-t="401">People, when they talk about this, they talk about it with the idea of like a labyrinth or a maze</a> <a data-t="407">or there's this paper that's called The Garden of Forking Paths.</a> <a data-t="412">The Garden of Forking Paths.</a> <a data-t="414">It's the idea that I'm walking through a garden, and there are all these paths that are like forking,</a> <a data-t="420">and I choose to go one way, and that means I would end up in a different place at the end</a> <a data-t="425">than if I chose to go a different direction.</a></p>

<p><a data-t="428">So when we are in this iterative process, this means that the decisions that we make,</a> <a data-t="435">as we're deciding, like as we're doing data analysis, they change where we may have ended up in the end.</a> <a data-t="441">So this paper specifically talks about this idea of a labyrinth or a garden of forking paths</a> <a data-t="451">in the context of how we use p-values in a statistical sense.</a> <a data-t="456">It talks about how dangerous it is to very naively use p-values when you've made a lot of decisions.</a></p>

<p><a data-t="463">But forking paths are a good thing.</a> <a data-t="466">Like it is good to make decisions and we base our next decision on the last one we made.</a> <a data-t="474">It is good to analyze data in different ways and to use what we learn about the data to do the next thing.</a> <a data-t="481">The mistake actually would be to choose just one, like I'm doing this, and I decided,</a> <a data-t="490">I don't care what I see in the data, that's just what I'm like.</a> <a data-t="493">I'm not going to use the information in the data itself.</a> <a data-t="496">That would be a mistake.</a> <a data-t="497">And another mistake is of the nature that is really focused on this paper,</a> <a data-t="502">would be, for example, to compute p-values without accounting for all the different choices we made.</a></p>

<p><a data-t="509">So the garden of forking paths is not a problem.</a> <a data-t="512">It is the only option we have to correctly analyze data.</a> <a data-t="517">But it means that as we move through a path, we end up at a place that would have been different</a> <a data-t="525">than if we took a different set of paths.</a> <a data-t="528">So this is part of why we have this very exploratory kind of iterative nature of data work.</a></p>
