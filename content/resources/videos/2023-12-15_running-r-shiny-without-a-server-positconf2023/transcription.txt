<h3>WebAssembly and the challenge of running R in the browser</h3>

<p><a data-t="137">And for a long time, that was a total pipe dream.</a> <a data-t="141">But a few years ago, I started to hear rumblings about this thing called WebAssembly, or Wasm.</a> <a data-t="147">And I'm going to give you the shortest description of Wasm that I can.</a></p>

<p><a data-t="152">What is Wasm?</a> <a data-t="153">Well, before Wasm, there was just JavaScript.</a> <a data-t="157">The browser only spoke one programming language, and it was JavaScript.</a> <a data-t="161">And it didn't matter how you felt about JavaScript.</a> <a data-t="163">If you wanted to target the browser, that was the language you had to write in.</a> <a data-t="167">And nowadays, browsers speak two languages.</a> <a data-t="170">There's still JavaScript, and now there's this new thing called Wasm.</a> <a data-t="173">But they are different because Wasm was designed for humans to write, and Wasm is designed</a> <a data-t="179">for compilers to emit.</a> <a data-t="181">So you don't write Wasm as a human being.</a> <a data-t="184">You write code in something like C or C++.</a> <a data-t="186">You compile it to Wasm, and that Wasm then runs in the browser.</a></p>

<p><a data-t="192">Now let's talk about why this is exciting.</a> <a data-t="196">And before we can, I have to come clean and admit to you all that I was wrong about Wasm,</a> <a data-t="202">that I was a huge skeptic of all this, and I'm going to explain why.</a></p>

<p><a data-t="207">And a lot of people within the company and some people outside of the company would hear</a> <a data-t="213">about Wasm, however they hear about things, and come to me or come to our team and say,</a> <a data-t="218">I have this great idea.</a> <a data-t="220">I have had this breakthrough in reasoning.</a> <a data-t="223">Shiny is written in R. R is written in C++.</a> <a data-t="227">C++ can be compiled to Wasm.</a> <a data-t="230">Therefore, Shiny can be compiled to Wasm QED.</a> <a data-t="234">Let me know when it ships.</a></p>

<p><a data-t="237">And this is a chain of reasoning that occurred independently to multiple people over multiple</a> <a data-t="242">years, and each time we would have this discussion, and I would have to disappoint them, and my</a> <a data-t="249">response eventually was like, well, actually, let me school you on what's happening here.</a></p>

<p><a data-t="256">Compiling C and C++ to Wasm is just the tip of a gigantic iceberg.</a> <a data-t="261">This is the biggest iceberg I could find on Unsplash, but if I could make the top smaller</a> <a data-t="265">and the bottom bigger, I would.</a> <a data-t="268">Because compiling C and C++ to Wasm is very exciting, but if you think about what R does</a> <a data-t="273">and what Python does, there's all this stuff under the surface beyond just interpreting</a> <a data-t="278">C and C++ instructions.</a></p>

<p><a data-t="281">There are all these things that are either from the operating system or from libraries</a> <a data-t="287">that we've built on over the years, and they are not available in Wasm.</a> <a data-t="294">And it's not just a matter of rebuilding those pieces that are missing, because the</a> <a data-t="298">environment that Wasm runs in is the browser, and the browser has a number of limitations</a> <a data-t="303">that your desktop computer is not subject to.</a> <a data-t="307">In the browser, you cannot launch a process, you can't open a network connection, you can't</a> <a data-t="311">even perform I.O., like reading from a disk or fetching from the network, synchronously.</a> <a data-t="317">You can't handle something as simple as Control-C.</a></p>

<p><a data-t="321">And Wasm does not lift any of these restrictions in and of itself, instead, it really just</a> <a data-t="328">makes it hurt more, because we have this software already written that's expecting to be able</a> <a data-t="332">to do these things, and we really wanted to just make it run in the browser, and we cannot</a> <a data-t="336">because these things make it so hard.</a></p>

<p><a data-t="340">And it makes me think of this Haitian proverb, de yin wan, yin wan, and what that means is</a> <a data-t="347">beyond mountains, more mountains.</a> <a data-t="351">Whatever hill you have to climb, whatever obstacle you're facing, whatever challenge</a> <a data-t="355">you overcome, when you get to the other side, it's just more of the same.</a></p>
