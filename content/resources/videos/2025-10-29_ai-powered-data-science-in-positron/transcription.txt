<h3>What is Positron?</h3>

<p><a data-t="32">All right so let's first talk about what is Positron.</a> <a data-t="37">For this demo we're going to define Positron as an AI-ready multilingual IDE for data scientists</a> <a data-t="44">but of course it can do a whole lot more.</a></p>

<p><a data-t="47">Now if you've never heard of Positron before maybe you've heard of </a><a href="/software/rstudio/">RStudio</a><a data-t="47">.</a> <a data-t="51">RStudio is an IDE that we developed at Posit probably 15 years ago at this point.</a> <a data-t="57">It's extremely stable, it's beloved by our users and it really is optimized for data science</a> <a data-t="64">because as you're working with data you're creating variables which you can actually</a> <a data-t="69">interact with in the variables pane and maybe you're creating plots or tables</a> <a data-t="73">which you can also interact with and see directly inside of the IDE itself.</a></p>

<p><a data-t="78">Now there's also another popular IDE it's called VS Code and while you can't do data</a> <a data-t="83">science in VS Code this tends to be an IDE that's extremely popular amongst software developers</a> <a data-t="90">because it is multilingual so you can use R, Python, Julia, HTML, JavaScript.</a> <a data-t="96">It's extremely customizable and in newer versions of VS Code it actually comes</a> <a data-t="101">AI-equipped with things like GitHub Copilot.</a></p>

<p><a data-t="103">So when designing Positron we really wanted to take the best of both worlds.</a> <a data-t="108">So we wanted to take the features that make RStudio optimize for data science and bring</a> <a data-t="113">that into Positron and then also bring in some of the cool features of VS Code</a> <a data-t="118">to create this perfect data science focused IDE.</a></p>

<h3>DataBot</h3>

<p><a data-t="478">Now, Positron Assistant is one tool that we're going to focus in on today.</a> <a data-t="482">But the other one is something called DataBot.</a> <a data-t="486">Now, DataBot is actually available as an extension.</a> <a data-t="489">So, it's not built into Positron like Positron Assistant.</a> <a data-t="492">It's available as an extension.</a></p>

<p><a data-t="502">But to talk a little bit about DataBot, I first want to help clarify, like, when to</a> <a data-t="508">use DataBot and when to use Positron Assistant.</a></p>

<p><a data-t="512">DataBot is specifically designed to do exploratory data analysis and to do it really fast.</a> <a data-t="519">And actually, it's surprisingly a lot of fun.</a> <a data-t="522">So, imagine you get some brand new data thrown at you and you want to analyze it.</a> <a data-t="526">That is a really good use case for DataBot.</a></p>

<p><a data-t="530">Now, for Positron Assistant, this can certainly do a lot of things that DataBot can do.</a> <a data-t="534">But it tends to be more task-focused.</a> <a data-t="536">So, maybe you have an error that you need to debug or you have a plot you want to approve</a> <a data-t="541">or you want to create, like, an individual table.</a> <a data-t="544">Those are really great tasks for Positron Assistant.</a></p>

<p><a data-t="548">Now, the way that DataBot works to do this exploratory data analysis is it implements</a> <a data-t="553">what we're calling this where loop.</a> <a data-t="556">So, you have some data and you want to learn more about that data.</a> <a data-t="560">So, you ask DataBot to start analyzing it.</a> <a data-t="563">And what it's going to do is first write some R or Python code.</a> <a data-t="566">It will execute that code immediately, analyze the results, regroup with suggestions,</a> <a data-t="572">and then just wait for your next instructions.</a> <a data-t="575">And this will just go around and around and around in circles until you're happy with</a> <a data-t="580">the results and you've gleaned all the information you need from that data.</a></p>

<p><a data-t="585">Now, it's also important to note that this where loop and also the way that Positron</a> <a data-t="589">Assistant works is that it's all code-based.</a> <a data-t="593">All the suggestions you're getting from DataBot are just code suggestions.</a> <a data-t="597">And we think that code is the most auditable way to explore data with AI.</a> <a data-t="601">Because you're not just asking it a question and then sending it off into some black box</a> <a data-t="605">and getting a magical response.</a> <a data-t="607">You are actually getting code.</a> <a data-t="609">You can see that code.</a> <a data-t="610">You can approve that code before it actually gets run.</a></p>

<blockquote><a data-t="597">And we think that code is the most auditable way to explore data with AI.</a> <a data-t="601">Because you're not just asking it a question and then sending it off into some black box</a> <a data-t="605">and getting a magical response.</a> <a data-t="607">You are actually getting code.</a> <a data-t="609">You can see that code.</a> <a data-t="610">You can approve that code before it actually gets run.</a></blockquote>
