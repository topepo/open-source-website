<h3>Positron Assistant</h3>

<p><a data-t="184">Positron Assistant is an AI tool that provides LLM integration within Positron. You can access</a> <a data-t="191">Positron Assistant from the Activity Bar. Use Positron Assistant to generate code, debug,</a> <a data-t="196">or get suggestions for next steps. Positron Assistant can access context from the Console,</a> <a data-t="203">Variables pane and Plots pane, but you can also manually attach files or folders to</a> <a data-t="208">give it additional context.</a></p>

<p><a data-t="213">Positron Assistant has three chat modes, Ask, Edit, and Agent. Use Ask mode when you want</a> <a data-t="219">to ask questions, get help with debugging, or use Assistant to generate code.</a></p>

<p><a data-t="232">For example, we could ask it what we should do next in our analysis of the diamonds data.</a> <a data-t="237">It can reference our </a><a href="/software/quarto/">Quarto</a><a data-t="237"> document and see what we've already done and suggest next steps.</a></p>

<p><a data-t="249">On Edit mode, Positron Assistant will suggest code changes.</a> <a data-t="255">So let's say I have another file and I'm getting an error. On Edit mode, I can ask Positron</a> <a data-t="261">Assistant to help fix the problem.</a></p>

<p><a data-t="268">And notice that it now knows that read diamonds.py is the current file.</a> <a data-t="277">Positron Assistant identifies the issue and supplies a fix that I can accept or undo.</a></p>

<p><a data-t="286">On Agent mode, Positron Assistant can figure out the next steps needed for your request</a> <a data-t="292">and autonomously carry out those steps. On your behalf, Positron Assistant can execute</a> <a data-t="297">code in the Console, create and modify files, and identify the next steps based on the results</a> <a data-t="302">of the previous steps. For example, we could ask it to move this code that's in read diamonds.py</a> <a data-t="308">into a new Quarto file.</a></p>
