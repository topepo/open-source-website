<h3>What is Quarto?</h3>

<p><a data-t="147">So let's start with learning a little bit about what Quarto is.</a> <a data-t="151">And Quarto is an open source technical publishing system.</a> <a data-t="156">You can think of it as sort of taking Jupyter notebooks, converting them to Markdown,</a> <a data-t="162">and then processing that Markdown into lots and lots and lots of different types of output formats and artifacts.</a></p>

<p><a data-t="163">Obviously, there are tools that do this, like nbconvert already,</a> <a data-t="167">but Quarto sort of takes that to a whole other level.</a> <a data-t="170">It supports not just the kind of conventional output formats,</a> <a data-t="174">like Word documents and PDFs and web pages and PowerPoint presentations and things like that,</a> <a data-t="179">but also specialized project types that let you build entire websites for books or blogs,</a> <a data-t="185">all from computations in Jupyter.</a></p>

<p><a data-t="187">So the real big idea behind Quarto is to sort of take the work that you do in Jupyter and in notebooks,</a> <a data-t="194">and then reproducibly transform it into various types of production output.</a> <a data-t="199">So again, Jupyter is where the computations happen.</a> <a data-t="203">Markdown is based on another open source project called Pandoc,</a> <a data-t="207">which is a very, very powerful kind of scientific markdown, academic and scientific markdown,</a> <a data-t="212">and supports lots of advanced features like citations and cross-references and things like that.</a> <a data-t="218">And again, turn it into lots of different types of output.</a></p>
