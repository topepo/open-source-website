<p><a href="/software/rstudio/">RStudio</a><a data-t="0"> Cloud allows us to create assignments for our learners via RStudio Projects.</a> <a data-t="8">By creating assignments within an RStudio Cloud workspace, we set up a shared environment</a> <a data-t="13">for our learners to access and complete their work.</a></p>

<p><a data-t="17">Assignments in RStudio Cloud happen at the project level.</a> <a data-t="21">To create a new assignment, we click on the New Project button here in the top right.</a> <a data-t="26">Once we're inside our project, we can create any files we'd like our students to use as</a> <a data-t="31">part of completing the assignment.</a> <a data-t="33">This could include an R Markdown document with instructions, an R script with partially</a> <a data-t="38">completed code, or any other type of file available to you from within RStudio.</a></p>

<h3>Making a project an assignment</h3>

<p><a data-t="45">After we create everything needed to get an assignment together, we'll want to make sure</a> <a data-t="49">that our learners can access it by clicking on the Project Settings button in the upper</a> <a data-t="53">right, clicking on the Access panel, and from the Who can view this project drop-down</a> <a data-t="58">menu, choosing Everyone.</a> <a data-t="61">The final step to making this an assignment is to check the box next to Make this project</a> <a data-t="65">an assignment.</a></p>

<p><a data-t="67">When we navigate back to our workspace by clicking on the workspace name at the top,</a> <a data-t="71">we see that our project is now listed as an assignment.</a> <a data-t="75">Our learners will see a corresponding tag that says either Start, if they haven't yet</a> <a data-t="79">opened the assignment, or Continue, if they've already opened the assignment.</a></p>

<p><a data-t="84">One thing to note is that if we make changes to an assignment, they will not be applied</a> <a data-t="88">to any copies of the assignment that students have already created by opening the assignment.</a> <a data-t="94">The changes we make to an assignment will only apply to future copies of the assignment.</a></p>
