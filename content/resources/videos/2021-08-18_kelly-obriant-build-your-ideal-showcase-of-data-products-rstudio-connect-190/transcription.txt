<h3>The content discovery problem</h3>

<p><a data-t="713">And I think this works like really well</a> <a data-t="718">for single items, at least.</a> <a data-t="720">If you are in a scenario where today you have Connect</a> <a data-t="724">and you only have like five different content items</a> <a data-t="728">and they're all beautiful,</a> <a data-t="730">you've set vanity URLs for them and they're all tagged,</a> <a data-t="734">maybe you don't even need tags, you only have five items.</a> <a data-t="736">So everything is labeled</a> <a data-t="738">and maybe your stakeholders have to do</a> <a data-t="740">a little bit of reading.</a> <a data-t="741">They've read the description,</a> <a data-t="743">associated metadata and thumbnail images with them.</a> <a data-t="749">Everything is fairly discoverable.</a> <a data-t="753">Content discovery is not an issue for you.</a></p>
