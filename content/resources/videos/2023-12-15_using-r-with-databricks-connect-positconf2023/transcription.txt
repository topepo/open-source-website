<h3>What is Spark Connect?</h3>

<p><a data-t="28">With Spark Connect, it's something that started</a> <a data-t="31">in a very, very recent version of Spark, 3.4.</a> <a data-t="35">In fact, it's so recent that 3.5 literally came out yesterday.</a> <a data-t="39">So we're talking about that close.</a> <a data-t="41">And with Spark Connect, it's not so much a new way of implementing,</a> <a data-t="44">like you have standalone or Kubernetes or Yarn.</a> <a data-t="48">This is more about how you connect to Spark, right?</a></p>

<p><a data-t="51">So if we're talking about a remote way of connecting to Spark</a> <a data-t="55">as it used to be before, that was through something called Livy.</a> <a data-t="58">And in order for us to use Livy,</a> <a data-t="60">we needed to have a full-on implementation</a> <a data-t="62">of a Yarn-based cluster, such as Hadoop cluster,</a> <a data-t="65">to be able to take advantage of that service,</a> <a data-t="69">which is not ideal in the new world that we live in,</a> <a data-t="72">where most of our data is now in S3 buckets, right?</a> <a data-t="75">All we need right now is not so much a place to land our data,</a> <a data-t="78">like Hadoop does, but more about how to process it, right?</a> <a data-t="81">A way to process it easily, which is through Spark, right?</a></p>
