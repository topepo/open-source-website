The shinythemes package provides Bootstrap themes from Bootswatch for Shiny applications. It has been superseded by the bslib package, which supports both Bootstrap 3 and 4.

This package offers 16 pre-built themes that can be applied to Shiny apps with a single argument. It includes a live theme selector tool that lets developers preview different themes interactively during development without reloading the app. The package solves the problem of quickly customizing the visual appearance of Shiny applications using standard Bootstrap CSS themes.
