asciicast converts R scripts into terminal screencasts in the asciinema format. It records all terminal output in real time, including interactive code execution, and can simulate realistic typing at configurable speeds.

The package generates v2 asciinema recordings that can be converted to SVG images for embedding in documentation, GitHub READMEs, or web pages. It captures errors and tracebacks accurately, supports both typed and instantly-displayed content blocks, and includes an HTML widget for use in R Markdown documents and vignettes. The tool requires no external dependencies for SVG conversion and works best in UTF-8 locales.
