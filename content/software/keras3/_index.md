Keras3 is an R interface to Keras, a high-level API for building and training neural networks. It enables R developers to prototype and implement deep learning models with minimal code, running seamlessly on either CPU or GPU.

The package provides a user-friendly API designed for fast experimentation, moving quickly from concept to working model. It includes built-in support for convolutional networks, recurrent networks, and custom architectures like multi-input/multi-output models and shared layers. Keras3 supports arbitrary network architectures, making it suitable for building any type of deep learning model.
